# EXPLANATION 17 2

**Question 2**

**Which prerequisites are required to use Volumes in Databricks?**

**Correct Answer:**\
✅ **Unity Catalog enabled and Databricks Runtime 13.3 LTS or later**

**Explanation**

Volumes require:

- Unity Catalog

- Databricks Runtime 13.3 LTS or later

Without Unity Catalog:

No Volumes

Databricks introduced Volumes as part of Unity Catalog governance, and support depends on compatible runtime versions. ([<u>docs.databricks.com</u>](https://docs.databricks.com/aws/en/volumes/?utm_source=chatgpt.com))
