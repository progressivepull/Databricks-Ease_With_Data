# EXPLANATION 17 5

**Question 5**

**Which DBUtils command is used to create directories inside a volume?**

**Correct Answer:**\
✅ **dbutils.fs.mkdirs()**

**Explanation**

Example:

dbutils.fs.mkdirs(\
"/Volumes/main/files/images"\
)

This creates directories exactly like Linux:

mkdir

Directory structure:

/Volumes\
\
└── main\
\
└── files\
\
└── images
