# EXPLANATION 18 3

**Question 2**

**Which DBUtils command displays all available DBUtils modules?**

**Correct Answer: C. dbutils.help()**

**Explanation**

Think of this as the "table of contents" for DBUtils.

dbutils.help()

Example output:

credentials\
fs\
jobs\
notebook\
secrets\
widgets

Each module has its own help.

Example:

dbutils.fs.help()

or

dbutils.widgets.help()

------------------------------------------------------------------------

**Why use help()?**

Suppose you forget the copy command.

Instead of Googling:

dbutils.fs.help()

Databricks lists every available filesystem command.
