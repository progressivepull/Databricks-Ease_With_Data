# EXPLANATION 18 1

**Databricks DBUtils Explained (Questions 1–10)**

DBUtils (Databricks Utilities) is a built-in utility library that helps you interact with the Databricks environment directly from notebooks. It provides commands for managing files, notebooks, widgets, secrets, jobs, and more. DBUtils is available in **Python, Scala, and R notebooks** (not SQL notebooks).
