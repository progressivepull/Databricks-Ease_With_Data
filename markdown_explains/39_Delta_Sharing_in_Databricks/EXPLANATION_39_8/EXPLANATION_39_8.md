# EXPLANATION 39 8

**Question 8**

**How does a Databricks recipient access shared data after it has been shared?**

**Correct Answer:** **B. By creating a Delta Sharing Catalog that references the provider's shared objects**

**Explanation**

After receiving access, the recipient creates a **Delta Sharing Catalog** in Unity Catalog. The shared tables, views, volumes, and functions appear inside that catalog and can be queried like local Unity Catalog objects.

**Why the Correct Answer is Correct**

The catalog provides a seamless way to browse and query shared assets.

**Why the Other Answers Are Wrong**

The incorrect options suggest importing or copying the data into the recipient's workspace.
