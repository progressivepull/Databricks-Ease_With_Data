# EXPLANATION 39 2

**Question 2**

**Which operation can a Delta Sharing recipient perform on shared data?**

**Correct Answer:** **D. Query and read shared objects without modifying them**

**Explanation**

Recipients receive **read-only** access to shared objects. They can:

- Run SQL queries.

- Read tables.

- Access shared volumes or functions (when granted).

They **cannot** insert, update, or delete the provider's data.

**Why the Correct Answer is Correct**

Delta Sharing is designed for secure collaboration while preserving the provider's ownership and integrity of the data.

**Why the Other Answers Are Wrong**

The incorrect options suggest recipients can modify, delete, or administer the shared data.
