# EXPLANATION 39 9

**Question 9**

**What authentication mechanism is typically used for Open Sharing?**

**Correct Answer:** **D. Sharing tokens**

**Explanation**

Open Sharing recipients authenticate using **sharing tokens**, which are delivered through a Delta Sharing credential file. The token securely authorizes access without requiring a Databricks account.

**Why the Correct Answer is Correct**

Sharing tokens provide secure authentication for external recipients using the Delta Sharing protocol.

**Why the Other Answers Are Wrong**

The incorrect options describe authentication methods used for Databricks workspaces rather than Open Sharing.
