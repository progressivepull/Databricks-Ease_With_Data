# EXPLANATION 39 3

**Question 3**

**Which Delta Sharing mode should be used when the recipient does not use Databricks?**

**Correct Answer:** **C. Open Sharing**

**Explanation**

Open Sharing is intended for recipients who are **not Databricks users**. Because Delta Sharing is an open protocol, recipients can use supported clients such as Apache Spark, pandas, Power BI, Tableau, and other compatible tools.

**Why the Correct Answer is Correct**

Open Sharing enables secure cross-platform sharing beyond Databricks workspaces.

**Why the Other Answers Are Wrong**

The incorrect options assume the recipient must already have a Databricks workspace.
