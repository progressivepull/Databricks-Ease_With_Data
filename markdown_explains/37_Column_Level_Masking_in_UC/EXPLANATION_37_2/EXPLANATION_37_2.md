# EXPLANATION 37 2

**Question 2**

**How does Column-Level Masking differ from Row-Level Security?**

**Correct Answer:** **B. Column-Level Masking protects individual column values, while Row-Level Security controls which rows are visible.**

**Explanation**

**Column-Level Masking**

- Controls what values users see within a column.

**Row-Level Security**

- Controls which rows users are allowed to see.

These features are complementary and are often used together.

**Why the Correct Answer is Correct**

They protect different dimensions of the data:

- Row-Level Security filters records.

- Column masking protects sensitive values.

**Why the Other Answers Are Wrong**

The incorrect options reverse the definitions or incorrectly state they perform the same function.
