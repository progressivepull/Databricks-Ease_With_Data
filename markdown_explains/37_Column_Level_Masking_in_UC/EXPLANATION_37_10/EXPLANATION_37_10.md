# EXPLANATION 37 10

**Question 10**

**Which combination of Unity Catalog security features provides a layered approach to data protection?**

**Correct Answer:** **B. Object-Level Security, Row-Level Security, and Column-Level Masking**

**Explanation**

Unity Catalog supports multiple layers of security:

- **Object-Level Security** – Controls access to catalogs, schemas, tables, and other securable objects.

- **Row-Level Security** – Controls which rows a user can see.

- **Column-Level Masking** – Controls what values users see within sensitive columns.

Together, these features provide comprehensive protection for both access and data visibility.

**Why the Correct Answer is Correct**

Each feature addresses a different security requirement:

- Object-Level Security → *Can I access this object?*

- Row-Level Security → *Which records can I see?*

- Column-Level Masking → *What values can I see?*

**Why the Other Answers Are Wrong**

The incorrect options omit one or more layers or combine unrelated features that do not provide end-to-end data protection.
