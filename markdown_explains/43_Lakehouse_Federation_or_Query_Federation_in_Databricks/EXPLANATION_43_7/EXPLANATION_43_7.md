# EXPLANATION 43 7

**Question 7**

**How are tables inside a Foreign Catalog identified?**

**Correct Answer:**\
**D. Type = Foreign**

**Why D is Correct**

When viewing tables inside Unity Catalog, federated tables are identified as:

Type = Foreign

This distinguishes them from native Delta tables while making it clear they reference an external data source.

**Why the other choices are wrong**

They are not labeled as:

- Delta

- Managed

- External (in the Unity Catalog table type sense)

The expected designation is **Foreign**.

**Memory Tip**

**Foreign Catalog → Foreign Tables**
