# EXPLANATION 43 6

**Question 6**

**Which compute environment can be used to test and run Lakehouse Federation connections?**

**Correct Answer:**\
**A. SQL Warehouses and Classic Databricks Compute**

**Why A is Correct**

Federated queries can run on:

- SQL Warehouses

- Classic Databricks Compute (clusters that support Unity Catalog and Federation)

This allows users to access external data from both SQL analytics and notebook-based workloads, subject to the supported runtime and configuration requirements.

**Why the other choices are wrong**

The other choices either omit supported compute types or include environments that are not appropriate for this lesson.
