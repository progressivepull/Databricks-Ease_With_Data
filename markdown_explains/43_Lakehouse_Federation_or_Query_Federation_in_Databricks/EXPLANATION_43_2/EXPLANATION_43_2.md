# EXPLANATION 43 2

**Question 2**

**Which of the following is a key benefit of using Lakehouse Federation?**

**Correct Answer:**\
**C. It reduces storage duplication by querying data where it already resides.**

**Why C is Correct**

Without Federation:

Database

↓

Copy Data

↓

Delta Table

↓

Query

With Federation:

Database

↑

Databricks Queries Directly

Benefits include:

- No duplicate storage

- Reduced ETL pipelines

- Fresher data

- Lower storage costs

**Why the other choices are wrong**

Federation does not primarily:

- improve ML accuracy

- replace Delta Lake

- eliminate Unity Catalog

- create dashboards

**Memory Tip**

**No Copies = Lower Cost**
