# EXPLANATION 43 5

**Question 5**

**Which of the following databases is supported by Lakehouse Federation?**

**Correct Answer:**\
**A. PostgreSQL**

**Why A is Correct**

PostgreSQL is one of the officially supported Lakehouse Federation data sources.

Other supported databases include:

- PostgreSQL

- MySQL

- SQL Server

- Oracle

- Snowflake

- Amazon Redshift

- Google BigQuery

- Azure Synapse

Support evolves over time, but PostgreSQL has been a supported option for Lakehouse Federation.

**Why the other choices are wrong**

Unsupported or incorrectly listed databases would not be valid choices for this question.

Think:
