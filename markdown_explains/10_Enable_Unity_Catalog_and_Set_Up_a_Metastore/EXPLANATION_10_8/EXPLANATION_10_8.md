# EXPLANATION 10 8

**Question 8**

**What additional step is required after creating a Metastore to enable Unity Catalog?**

**Correct Answer:**\
✅ **Assign the Metastore to a Databricks workspace**

**Explanation**

Creating the Metastore alone is not enough.

You must attach it to one or more workspaces.

Workflow:

Create Metastore\
\
↓\
\
Assign Workspace\
\
↓\
\
Refresh Workspace\
\
↓\
\
Unity Catalog Available

Once assigned, users in that workspace can:

- Create catalogs

- Create schemas

- Govern data

- Use centralized permissions
