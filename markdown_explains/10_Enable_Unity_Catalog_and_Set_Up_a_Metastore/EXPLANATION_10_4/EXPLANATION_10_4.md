# EXPLANATION 10 4

**Question 4**

**Which Azure feature must be enabled to use Azure Data Lake Storage Gen2 (ADLS Gen2)?**

**Correct Answer:**\
✅ **Hierarchical Namespace**

**Explanation**

A Storage Account becomes **Azure Data Lake Storage Gen2** when **Hierarchical Namespace (HNS)** is enabled.

Without HNS:

Blob Storage

With HNS:

Azure Data Lake Storage Gen2

Hierarchical Namespace enables:

- Folder structure

- File-level operations

- Better analytics performance

- Hadoop-compatible filesystem semantics

It is required for ADLS Gen2 functionality. Microsoft identifies Hierarchical Namespace as the defining feature of ADLS Gen2.
