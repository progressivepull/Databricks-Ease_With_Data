# EXPLANATION 10 10

**Question 10**

**Which new capability becomes available after Unity Catalog is successfully enabled and the workspace is refreshed?**

**Correct Answer:**\
✅ **Catalog creation and centralized workspace permissions**

**Explanation**

Once Unity Catalog is enabled and attached to a workspace, users can begin creating governed objects.

New capabilities include:

- Create catalogs

- Create schemas

- Apply centralized permissions

- Share data across workspaces

- Fine-grained access control

- Data lineage

- Auditing

Example:

Workspace\
\
↓\
\
Create Catalog\
\
↓\
\
Create Schema\
\
↓\
\
Create Tables\
\
↓\
\
Grant Permissions

Instead of managing permissions separately in each workspace, administrators define them once in Unity Catalog and apply them consistently across attached workspaces.
