# EXPLANATION 10 9

**Question 9**

**Who automatically becomes the Metastore Administrator after creating the Metastore?**

**Correct Answer:**\
✅ **The user who created the Metastore**

**Explanation**

The creator automatically becomes the first **Metastore Administrator**.

The Metastore Administrator can:

- Create catalogs

- Manage permissions

- Delegate admin roles

- Configure storage

- Manage governance settings

Additional administrators can be granted later.

**Example**

Create Metastore\
\
↓\
\
John\
\
↓\
\
Metastore Administrator
