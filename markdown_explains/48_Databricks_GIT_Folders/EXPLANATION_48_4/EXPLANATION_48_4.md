# EXPLANATION 48 4

**Question 4**

**How does Databricks authenticate with Azure DevOps when connecting a Git repository?**

**Correct Answer:**\
**B. Username and Personal Access Token (PAT)**

**Why B is Correct**

Databricks authenticates to Azure DevOps by using:

- Username (or account identifier)

- Personal Access Token (PAT)

The PAT acts like a password with configurable scopes, allowing secure access without using your actual account password. Databricks also supports other Git provider authentication methods depending on the provider.

**Why the other choices are wrong**

Databricks does not typically authenticate to Azure DevOps Git repositories with SQL credentials or Spark credentials.

**Memory Tip**

**Azure DevOps → PAT**
