# EXPLANATION 48 2

**Question 2**

**Which Git operation is supported by Databricks Git Folders?**

**Correct Answer:**\
**A. Clone repositories**

**Why A is Correct**

The first step when using Git Folders is usually:

Remote Repository

↓

Clone

↓

Git Folder

Cloning downloads the remote repository into your Databricks workspace while preserving its Git history. Git Folders also support commit, push, pull, merge, rebase, and branch operations.

**Why the other choices are wrong**

Unsupported answers typically describe actions outside standard Git operations.

**Memory Tip**

**Clone = First Step**
