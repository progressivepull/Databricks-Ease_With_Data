# EXPLANATION 48 3

**Question 3**

**Which Azure DevOps service is used to store notebooks, source code, and configuration files using Git version control?**

**Correct Answer:**\
**C. Repos**

**Why C is Correct**

Azure DevOps **Repos** is Microsoft's Git repository hosting service.

It stores:

- Python files

- SQL files

- Notebooks

- Configuration files

- Documentation

- CI/CD definitions

Databricks Git Folders connect directly to Azure DevOps Repos using Git.

**Why the other choices are wrong**

- **Pipelines** automate builds and deployments.

- **Boards** manage work items.

- **Artifacts** store packages.

**Memory Tip**

**Azure DevOps Repos = Git Storage**
