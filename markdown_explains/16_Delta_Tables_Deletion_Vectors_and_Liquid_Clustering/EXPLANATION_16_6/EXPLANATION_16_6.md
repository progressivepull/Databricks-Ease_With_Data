# EXPLANATION 16 6

**Question 6**

**Which Delta Lake and Databricks Runtime versions are required to use Deletion Vectors?**

**Correct Answer (for your lesson):**\
✅ **Delta Lake 2.3+ and Databricks Runtime 12.2 LTS+**

**Explanation**

Deletion Vectors were introduced beginning with **Delta Lake 2.3**. Your course correctly associates them with **Databricks Runtime 12.2 LTS+** for Azure Databricks learning.

Since then, Databricks has expanded and enhanced support in newer runtimes, and current documentation recommends using the latest supported Databricks Runtime for the best experience.
