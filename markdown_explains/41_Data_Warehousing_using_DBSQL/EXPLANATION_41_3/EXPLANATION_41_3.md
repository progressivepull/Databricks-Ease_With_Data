# EXPLANATION 41 3

**Question 3**

**Which type of SQL Warehouse is recommended for most workloads?**

**Correct Answer:**\
**C. Serverless**

**Why C is Correct**

Databricks recommends Serverless SQL Warehouses because they:

- start in seconds

- automatically scale

- require no infrastructure management

- optimize cost and performance

- simplify administration

Users simply run queries while Databricks manages the infrastructure.

**Why the others are wrong**

Classic and Pro warehouses still work, but they require more management or are intended for specialized scenarios.

**Memory Tip**

**New projects → Serverless**
