# EXPLANATION 41 6

**Question 6**

**How do SQL Warehouses improve support for many simultaneous users?**

**Correct Answer:**\
**C. By using multiple clusters and Intelligent Workload Management to distribute concurrent queries**

**Why C is Correct**

Serverless SQL Warehouses can:

- start multiple compute clusters

- automatically balance concurrent queries

- prevent one long-running query from blocking others

This provides high concurrency for dashboards and BI users.

**Why the others are wrong**

They do not simply increase CPU on one machine or queue every query sequentially.

**Memory Tip**

**Many users → Multiple clusters**
