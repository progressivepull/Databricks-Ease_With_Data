# EXPLANATION 41 8

**Question 8**

**What information can users analyze in the Query Profile after executing a SQL query?**

**Correct Answer:**\
**B. Query execution plan, execution time, memory usage, rows processed, and data scanned**

**Why B is Correct**

The Query Profile helps diagnose query performance by showing:

- execution plan

- query stages

- execution time

- rows processed

- data scanned

- memory and resource usage

This information helps identify bottlenecks and optimize queries.

**Why the others are wrong**

The Query Profile is for performance analysis, not permissions or schema management.
