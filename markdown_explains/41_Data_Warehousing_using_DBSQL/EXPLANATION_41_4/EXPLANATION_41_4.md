**Question 4**

**Which optimization technology is used by Serverless SQL Warehouses to accelerate SQL query execution?**

**Correct Answer:**\
**B. Photon Engine**

**Why B is Correct**

Photon is Databricks' high-performance vectorized execution engine.

It is designed specifically to speed up SQL and DataFrame operations by:

- using modern CPU instructions

- reducing execution overhead

- processing data in vectorized batches

This often results in significantly faster query execution than the traditional Spark engine.

**Why the others are wrong**

None of the other options are Databricks' SQL execution engine.
