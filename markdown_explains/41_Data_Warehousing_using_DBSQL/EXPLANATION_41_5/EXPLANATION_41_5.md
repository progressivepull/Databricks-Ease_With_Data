# EXPLANATION 41 5

**Question 5**

**What is the primary benefit of Predictive I/O in Serverless SQL Warehouses?**

**Correct Answer:**\
**B. Faster data scanning and reduced query latency**

**Why B is Correct**

Predictive I/O intelligently predicts which data blocks will be needed and optimizes data access. Benefits include:

- fewer storage reads

- faster scans

- lower query latency

- improved analytics performance

This is especially useful for large Delta tables.

**Why the others are wrong**

Predictive I/O is not responsible for:

- security

- cluster creation

- dashboard generation

- BI connectivity
