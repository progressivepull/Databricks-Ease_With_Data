# EXPLANATION 41 9

**Question 9**

**What capability does the SQL Editor provide for saved queries?**

**Correct Answer:**\
**B. Scheduling queries to run automatically for recurring reports and dashboards**

**Why B is Correct**

Saved SQL queries can be scheduled to:

- refresh reports

- update dashboards

- populate downstream workflows

- run at recurring intervals

This eliminates the need to execute routine queries manually.

**Why the others are wrong**

Scheduling is not related to machine learning, cluster management, or Delta table creation.

**Memory Tip**

**Save → Schedule → Automate**
