**Question 1**

**What is the primary purpose of Databricks SQL (DBSQL)?**

**Correct Answer:**\
**B. To provide a SQL analytics service for querying data, building reports, dashboards, and connecting BI tools**

**Why B is Correct**

Databricks SQL (DBSQL) is the analytics and data warehousing component of the Databricks Lakehouse Platform. It allows analysts and business users to:

- Run ANSI SQL queries

- Analyze Delta Lake tables

- Create reports

- Build dashboards

- Schedule SQL queries

- Connect BI tools such as Power BI, Tableau, and Looker

Unlike notebooks primarily used by engineers, DBSQL is optimized for SQL analytics and business intelligence.

**Why the other choices are wrong**

- **A.** DBSQL is not a data ingestion service.

- **C.** It does not replace Spark; Spark remains the execution engine.

- **D.** It is not used for machine learning model training.

- **E.** It does not replace Unity Catalog.
