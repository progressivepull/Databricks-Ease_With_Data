# EXPLANATION 41 7

**Question 7**

**Which information is typically required for a BI tool such as Power BI or Tableau to connect to a SQL Warehouse?**

**Correct Answer:**\
**C. Hostname, HTTP Path, and JDBC URL**

**Why C is Correct**

Most BI tools require connection information including:

- Server Hostname

- HTTP Path

- JDBC or ODBC connection details

- Authentication credentials or tokens

These identify the SQL Warehouse and allow secure connections.

**Why the others are wrong**

Cluster IDs or notebook paths are not sufficient for BI connectivity.

**Memory Tip**

Remember:

- Hostname

- HTTP Path

- JDBC/ODBC
