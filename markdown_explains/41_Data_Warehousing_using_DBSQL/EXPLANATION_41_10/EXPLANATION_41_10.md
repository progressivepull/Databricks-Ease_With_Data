# EXPLANATION 41 10

**Question 10**

**Which statement correctly compares SQL Warehouses with All-Purpose Clusters?**

**Correct Answer:**\
**A. SQL Warehouses are optimized for analytics (OLAP), BI connectivity, and concurrent SQL workloads, while All-Purpose Clusters are designed primarily for engineering and development.**

**Why A is Correct**

**SQL Warehouses** are designed for:

- SQL analytics

- dashboards

- BI tools

- high concurrency

- business reporting

**All-Purpose Clusters** are designed for:

- notebooks

- Spark development

- Python

- Scala

- machine learning

- data engineering

Each compute type is optimized for different workloads.

**Why the others are wrong**

Neither compute type replaces the other. SQL Warehouses specialize in analytics, while All-Purpose Clusters provide a general-purpose development environment.

**Memory Tip**

| **SQL Warehouse** | **All-Purpose Cluster**   |
|-------------------|---------------------------|
| SQL Analytics     | Engineering & Development |
| BI Tools          | Python / Scala / Spark    |
| Dashboards        | ETL & Machine Learning    |
| High Concurrency  | Interactive Development   |
