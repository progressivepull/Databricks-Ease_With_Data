**Question 2**

**What is the role of a SQL Warehouse in Databricks?**

**Correct Answer:**\
**C. It serves as the compute engine that executes SQL queries.**

**Why C is Correct**

A SQL Warehouse is the compute resource dedicated to SQL workloads. Every SQL query, dashboard refresh, or BI connection runs against a SQL Warehouse.

Think of it as the SQL equivalent of a Spark cluster, optimized specifically for analytics.

**Why the others are wrong**

- It does not permanently store data.

- It is not Unity Catalog.

- It does not replace Delta Lake.

- It is not a visualization tool.
