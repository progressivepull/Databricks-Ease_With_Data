# EXPLANATION 25 7

**Question 7**

**Which file detection mode is the default behavior?**

**Correct Answer**

✅ **D. Directory Listing**

**Explanation**

Auto Loader supports two discovery modes:

**1. Directory Listing (Default)**

- scans folders

- finds new files

- simple

- no cloud notifications required

**2. File Notification**

- uses Azure Event Grid

- AWS SQS/SNS

- GCP Pub/Sub

Databricks recommends file notification for many production workloads, but **directory listing remains the default mode**.

**Easy Memory Trick**

Default = Directory Listing
