**Question 4**

**Which type of information does Genie primarily use to understand the meaning of user questions?**

**Correct Answer:**\
**C. Metadata such as table descriptions, column comments, dataset names, and relationships**

**Why C is Correct**

Genie relies heavily on metadata, including:

- table descriptions

- column comments

- business-friendly dataset names

- relationships between datasets

- semantic context

Good metadata greatly improves the quality of generated SQL and answers.

**Why the other choices are wrong**

Genie is not trained specifically on:

- notebook code

- cluster configurations

- Spark logs
