# EXPLANATION 47 8

**Question 7**

**What does the Show Code option allow users to do?**

**Correct Answer:**\
**C. Inspect the generated SQL, including filters, joins, and aggregations**

**Why C is Correct**

Selecting **Show Code** reveals the SQL generated by Genie.

Users can inspect:

- joins

- filters

- GROUP BY clauses

- aggregations

- calculated fields

This improves transparency and helps analysts validate or learn from the generated query.

**Why the other choices are wrong**

Show Code does not expose:

- Python notebooks

- Spark execution logs

- cluster configuration

**Memory Tip**

**Show Code = See the SQL Genie wrote.**
