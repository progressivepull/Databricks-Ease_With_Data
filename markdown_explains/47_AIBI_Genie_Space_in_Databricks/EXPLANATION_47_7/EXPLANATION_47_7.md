# EXPLANATION 47 7

**Question 6**

**Why would you add custom instructions to a Genie Space?**

**Correct Answer:**\
**B. To teach Genie organization-specific terminology, abbreviations, and business language**

**Why B is Correct**

Custom instructions let administrators explain organization-specific terms.

Examples:

- PO = Purchase Order

- ARR = Annual Recurring Revenue

- Customer = Active Subscriber

These instructions improve the accuracy of generated SQL and responses.

**Why the other choices are wrong**

Instructions do not:

- increase warehouse size

- create dashboards

- modify Delta tables
