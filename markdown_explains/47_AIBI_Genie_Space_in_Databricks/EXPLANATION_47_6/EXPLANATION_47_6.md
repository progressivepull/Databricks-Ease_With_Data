# EXPLANATION 47 6

**Question 5**

**What is the purpose of configuring Example Values in a Genie Space?**

**Correct Answer:**\
**B. To improve AI interpretation of user questions by providing representative data values**

**Why B is Correct**

Example Values help Genie understand business terminology.

For example:

Region

East

West

North

South

or

Status

Pending

Approved

Cancelled

Providing representative values helps Genie correctly interpret user questions such as "Show pending orders" or "Sales in the West region."

**Why the other choices are wrong**

Example Values are not used to:

- speed up SQL execution

- create indexes

- improve cluster performance

**Memory Tip**

**Examples teach the AI your business vocabulary.**
