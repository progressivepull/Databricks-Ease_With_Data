# EXPLANATION 47 2

**Question 2**

**When a user asks a question in plain English, what sequence of actions does Genie perform?**

**Correct Answer:**\
**B. Generates SQL, executes it on a SQL Warehouse, and returns results with tables and visualizations**

**Why B is Correct**

The workflow is:

Natural Language Question

↓

AI interprets the request

↓

Generate SQL

↓

Execute on SQL Warehouse

↓

Return table

↓

Generate visualization

This allows business users to obtain insights without knowing SQL.

**Why the other choices are wrong**

Genie does not:

- execute Python notebooks

- bypass the SQL Warehouse

- manually ask users to write SQL first
