# EXPLANATION 47 1

**Question 1**

**What is the primary purpose of AI/BI Genie Spaces in Databricks?**

**Correct Answer:**\
**B. To allow business users to query data using natural language instead of writing SQL**

**Why B is Correct**

AI/BI Genie Spaces provide a conversational interface where users can ask questions such as:

- "What were total sales last month?"

- "Which products had the highest revenue?"

- "Show me orders by region."

Instead of writing SQL, users ask questions in plain English. Genie translates the request into SQL, executes it, and returns results grounded in governed data.

**Why the other choices are wrong**

Genie Spaces do **not**:

- replace SQL Warehouses

- build ETL pipelines

- train machine learning models

- replace Unity Catalog
