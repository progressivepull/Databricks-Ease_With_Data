# EXPLANATION 47 4

**Question 3**

**Which SQL Warehouse types are supported by AI/BI Genie?**

**Correct Answer:**\
**B. SQL Warehouse Pro and SQL Warehouse Serverless**

**Why B is Correct**

Genie requires a supported SQL Warehouse to execute generated SQL.

Supported types include:

- SQL Warehouse Pro

- SQL Warehouse Serverless

A warehouse with at least **CAN USE** permission must be configured for the Genie Space.

**Why the other choices are wrong**

Genie does not execute queries directly on:

- All-Purpose clusters

- Jobs clusters

- ML clusters

**Memory Tip**

**Genie runs on SQL Warehouses.**
