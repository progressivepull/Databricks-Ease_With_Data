# EXPLANATION 08 6

**Question 6**

**Which of the following objects can be stored inside a schema?**

**Correct Answer:**\
✅ **Tables, Views, Functions, and Volumes**

**Explanation**

A schema organizes related database objects.

Example:

Sales Schema\
\
├── Orders\
\
├── Customers\
\
├── Sales View\
\
├── Functions\
\
└── Volume

A schema is similar to a folder that groups related objects together.
