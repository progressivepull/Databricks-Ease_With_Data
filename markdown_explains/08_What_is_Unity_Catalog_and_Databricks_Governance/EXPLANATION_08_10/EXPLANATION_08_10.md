# EXPLANATION 08 10

**Question 10**

**Why is centralized governance with Unity Catalog beneficial for organizations?**

**Correct Answer:**\
✅ **It provides consistent security, simplified administration, auditing, and governance across multiple workspaces.**

**Explanation**

Large organizations often have:

- Many workspaces

- Thousands of users

- Hundreds of data engineers

- Hundreds of data scientists

Without centralized governance:

Workspace A\
\
Security\
\
Workspace B\
\
Security\
\
Workspace C\
\
Security

Administrators must manage everything separately.

With Unity Catalog:

Unity Catalog\
\
↓\
\
Users\
\
↓\
\
Groups\
\
↓\
\
Permissions\
\
↓\
\
All Workspaces

Benefits include:

- Centralized permissions

- Fine-grained access control

- Data lineage

- Auditing

- Regulatory compliance

- Easier administration

- Consistent governance

This greatly reduces administrative effort while improving security.
