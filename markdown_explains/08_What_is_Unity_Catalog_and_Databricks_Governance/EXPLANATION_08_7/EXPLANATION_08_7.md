# EXPLANATION 08 7

**Question 7**

**What types of data can Unity Catalog Volumes store?**

**Correct Answer:**\
✅ **Structured, semi-structured, and unstructured data**

**Explanation**

Volumes are designed for files.

Examples include:

**Structured**

- CSV

- Parquet

**Semi-Structured**

- JSON

- XML

- Avro

**Unstructured**

- Images

- PDFs

- Videos

- ZIP files

- Machine Learning models

Unlike Delta tables, Volumes are intended for storing general files under Unity Catalog governance.

**Example**

Volume\
\
├── invoices.csv\
\
├── photo.png\
\
├── report.pdf\
\
├── model.pkl\
\
└── data.json
