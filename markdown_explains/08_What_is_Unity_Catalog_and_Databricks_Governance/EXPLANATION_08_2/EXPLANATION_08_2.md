**Question 2**

**What is the core principle of Unity Catalog?**

**Correct Answer:**\
✅ **Define security once and enforce it everywhere.**

**Explanation**

This is the biggest benefit of Unity Catalog.

Without Unity Catalog:

Workspace A\
\
Permissions\
\
Workspace B\
\
Permissions\
\
Workspace C\
\
Permissions

Every workspace requires separate permission management.

With Unity Catalog:

Unity Catalog\
\
↓\
\
Permissions\
\
↓\
\
All Workspaces

One permission definition applies everywhere.

Benefits:

- Consistent security

- Easier administration

- Fewer mistakes

- Better compliance
