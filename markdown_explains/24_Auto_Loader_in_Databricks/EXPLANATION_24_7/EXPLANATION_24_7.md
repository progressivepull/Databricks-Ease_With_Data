# EXPLANATION 24 7

**Question 7**

**Which file detection mode is the default behavior for Auto Loader?**

**Correct Answer:** **D. Directory Listing**

**Explanation**

Auto Loader supports two discovery modes:

1.  **Directory Listing** (default)

2.  **File Notification**

Directory Listing periodically scans the source directory to discover new files. Databricks recommends file notification mode for many production workloads because it can reduce file discovery costs, but Directory Listing remains the default behavior.
