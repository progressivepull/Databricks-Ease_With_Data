# EXPLANATION 24 8

**Question 8**

**Which Auto Loader option enables event-based file detection using cloud notification services?**

**Correct Answer:** **B. cloudFiles.useNotifications=true**

**Explanation**

Instead of repeatedly scanning directories, Auto Loader can receive cloud storage notifications when new files arrive.

Enable it with:

.option("cloudFiles.useNotifications", "true")

Benefits include:

- Faster file discovery

- Lower cloud API costs

- Better scalability for large workloads
