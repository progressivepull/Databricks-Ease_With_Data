# EXPLANATION 24 5

**Question 5**

**Which Auto Loader option specifies where inferred schemas are stored?**

**Correct Answer:** **C. cloudFiles.schemaLocation**

**Explanation**

Auto Loader automatically infers schemas and stores them in a schema tracking location specified by:

.option("cloudFiles.schemaLocation", "/schemas/orders")

This location tracks schema changes over time so Auto Loader can evolve the schema safely.
