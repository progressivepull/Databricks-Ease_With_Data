# EXPLANATION 40 2

**Question 2**

**Which task is still the responsibility of a user when using Serverless Compute?**

**Correct Answer:**

**C. Applying Serverless to the workload and running the code**

**Why C is Correct**

Although Databricks manages infrastructure, users still decide:

- which notebook to run

- which job to execute

- whether to use Serverless

- what code to write

Databricks **does not write or execute your application automatically**.

------------------------------------------------------------------------

**Why the others are wrong**

Infrastructure management is handled automatically:

- Cluster creation ✔

- Scaling ✔

- VM management ✔

- Runtime updates ✔

Only workload selection remains your responsibility.

------------------------------------------------------------------------

**Memory Tip**

Serverless removes infrastructure work—not application development.
