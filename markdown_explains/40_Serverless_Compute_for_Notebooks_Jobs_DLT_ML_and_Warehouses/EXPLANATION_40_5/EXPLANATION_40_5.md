# EXPLANATION 40 5

**Question 5**

**What happens when a Serverless Compute workload requires additional resources?**

**Correct Answer:**

**C. Serverless automatically scales resources up or down as needed.**

------------------------------------------------------------------------

**Why C is Correct**

Autoscaling is built into Serverless.

As workload increases:

- More compute is added.

As workload decreases:

- Compute is removed automatically.

------------------------------------------------------------------------

**Why the others are wrong**

Users do **not** manually:

- add workers

- resize clusters

- restart clusters

------------------------------------------------------------------------

**Memory Tip**

Serverless = Elastic Compute.
