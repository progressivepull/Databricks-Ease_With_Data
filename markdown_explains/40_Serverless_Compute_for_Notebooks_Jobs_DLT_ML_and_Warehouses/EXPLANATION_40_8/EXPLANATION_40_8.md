# EXPLANATION 40 8

**Question 8**

**Which Databricks workloads can use Serverless Compute?**

**Correct Answer:**

**C. Notebooks, Workflows (Jobs), Delta Live Tables (DLT), Machine Learning workloads, and SQL Warehouses**

------------------------------------------------------------------------

**Why C is Correct**

Serverless supports many Databricks services, including:

- Notebooks

- Jobs (Workflows)

- Lakeflow/Delta Live Tables

- Machine Learning workloads

- SQL Warehouses (through separate serverless SQL infrastructure)

------------------------------------------------------------------------

**Why the others are wrong**

Other answers omit supported workloads.

------------------------------------------------------------------------

**Memory Tip**

Think:

Almost every major Databricks workload can use Serverless.
