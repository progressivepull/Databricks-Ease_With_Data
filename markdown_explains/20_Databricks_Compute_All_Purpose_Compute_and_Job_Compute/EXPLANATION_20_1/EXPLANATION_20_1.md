**Databricks Compute (Clusters) Explained (Questions 1–10)**

A **Databricks Compute**, often called a **cluster**, is a collection of virtual machines that runs Apache Spark. It provides the processing power needed to execute notebooks, SQL queries, machine learning workloads, streaming jobs, and workflows.

Think of a cluster as the "engine" that performs the work, while your notebook contains the instructions.
