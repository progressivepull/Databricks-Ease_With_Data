# EXPLANATION 20 6

**Question 5**

**Which Access Mode supports multiple users while remaining compatible with Unity Catalog?**

**Correct Answer: C. Shared**

**Explanation**

Access Modes determine how users interact with a cluster.

Common modes:

- **Single User** – Dedicated to one user.

- **Shared** – Multiple users can use the same cluster while remaining compatible with Unity Catalog.

- **No Isolation Shared** – Legacy option with reduced isolation (not recommended for new workloads).

The **Shared** access mode provides:

- Multi-user collaboration.

- Unity Catalog support.

- Improved security and governance.

This is the recommended choice for many collaborative environments.
