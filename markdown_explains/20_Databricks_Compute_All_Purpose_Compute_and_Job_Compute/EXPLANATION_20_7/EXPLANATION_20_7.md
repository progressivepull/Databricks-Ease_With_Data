# EXPLANATION 20 7

**Question 6**

**What is the recommendation when selecting a Databricks Runtime (DBR) version?**

**Correct Answer: B. Use the latest Long-Term Support (LTS) version whenever possible.**

**Explanation**

The Databricks Runtime (DBR) includes:

- Apache Spark.

- Optimizations.

- Libraries.

- Security fixes.

- Bug fixes.

An **LTS (Long-Term Support)** release is recommended because it:

- Receives extended support.

- Is more stable.

- Includes important security updates.

- Minimizes compatibility issues.

Use a non-LTS version only when you need features not yet available in the latest LTS release.
