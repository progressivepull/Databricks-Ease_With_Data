# EXPLANATION 20 8

**Question 7**

**What is the primary benefit of enabling Photon?**

**Correct Answer: C. It improves ETL, SQL, streaming, and data science performance through a high-performance execution engine.**

**Explanation**

Photon is Databricks' native high-performance query engine.

It accelerates workloads such as:

- SQL analytics.

- ETL pipelines.

- Delta Lake operations.

- DataFrame transformations.

- Streaming.

- Data science workloads.

Benefits include:

- Faster execution.

- Lower latency.

- Reduced cloud costs.

- Better CPU utilization.

Photon often improves performance without requiring any changes to your notebook code.
