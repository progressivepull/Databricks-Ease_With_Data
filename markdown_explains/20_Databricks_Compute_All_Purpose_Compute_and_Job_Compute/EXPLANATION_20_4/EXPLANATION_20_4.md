# EXPLANATION 20 4

**Question 3**

**Which type of compute is best suited for interactive notebook development and manual execution?**

**Correct Answer: C. All-Purpose Compute**

**Explanation**

All-Purpose Compute is designed for interactive work by developers, analysts, and data scientists.

Typical uses include:

- Developing notebooks.

- Testing code.

- Exploring datasets.

- Debugging pipelines.

- Ad hoc SQL analysis.

Characteristics:

- Started manually or kept running.

- Shared among team members (depending on permissions).

- Optimized for interactive sessions.

Example workflow:

1.  Start cluster.

2.  Open notebook.

3.  Write Spark code.

4.  Run cells repeatedly.

5.  Stop cluster when finished.

**Why not Job Compute?**

Job Compute is intended for automated execution, not interactive development.
