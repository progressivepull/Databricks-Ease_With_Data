# EXPLANATION 35 2

**Question 2**

**Which of the following is an example of a built-in Databricks SQL function?**

**Correct Answer:** **C. LOWER()**

**Explanation**

LOWER() is a built-in ANSI SQL function supported by Databricks SQL. It converts every character in a string to lowercase.

Example:

SELECT LOWER('Databricks');

Output:

databricks

**Why the Correct Answer is Correct**

LOWER() is included with Databricks SQL and requires no custom implementation.

**Why the Other Answers Are Wrong**

The other options are either not built-in SQL functions or are fictitious function names.
