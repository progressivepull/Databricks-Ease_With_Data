# EXPLANATION 35 10

**Question 10**

**Which statement correctly describes function parameters and function types?**

**Correct Answer:** **D. The output determines the function type: one value is a scalar function, while multiple rows form a table function.**

**Explanation**

Function parameters define the inputs to a function, but **the function's output determines its type**:

- **Scalar function** → returns a single value.

- **Table function** → returns a result set with multiple rows (and potentially multiple columns).

**Why the Correct Answer is Correct**

The defining characteristic is the return value, not the number of parameters or the implementation language.

**Why the Other Answers Are Wrong**

The incorrect options confuse input parameters with return types or incorrectly classify functions based on how they are written.
