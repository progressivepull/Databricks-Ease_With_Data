# EXPLANATION 35 5

**Question 5**

**What is the benefit of using CREATE OR REPLACE FUNCTION?**

**Correct Answer:** **B. It allows a function to be updated without manually dropping it first.**

**Explanation**

CREATE OR REPLACE FUNCTION replaces an existing function definition if it already exists. This simplifies development because developers do not need to explicitly drop the old function before creating the new version.

**Why the Correct Answer is Correct**

Benefits include:

- Easier development.

- Faster iteration.

- Simpler deployment scripts.

- Safer CI/CD automation.

**Why the Other Answers Are Wrong**

They incorrectly state that the function must always be manually deleted first.
