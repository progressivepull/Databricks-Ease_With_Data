# EXPLANATION 35 9

**Question 9**

**Which information can be viewed for a registered function in Catalog Explorer?**

**Correct Answer:** **C. Input parameters, return type, function type, language, and definition**

**Explanation**

Catalog Explorer displays detailed metadata for registered functions, including:

- Function name

- Input parameters

- Return type

- Function type (scalar or table)

- Programming language

- SQL or Python definition

- Comments and documentation, when provided.

**Why the Correct Answer is Correct**

Catalog Explorer is designed to help users discover, understand, and reuse registered functions.

**Why the Other Answers Are Wrong**

The other options omit important metadata or include information unrelated to function definitions.
