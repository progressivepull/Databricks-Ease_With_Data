# EXPLANATION 02 2

**Artificial Intelligence (AI)**

**What is Artificial Intelligence (AI)?**

Artificial Intelligence (AI) is the ability of machines or software systems to perform tasks that normally require human intelligence.

These tasks include:

- learning

- reasoning

- problem-solving

- understanding language

- recognizing images

- making decisions

In simple terms:

AI enables computers to think and act in smart ways.

------------------------------------------------------------------------

**Simple Real-World Example**

When Netflix recommends movies, or Google Maps suggests the fastest route, or ChatGPT answers questions — AI is being used.

AI systems learn from:

- data

- patterns

- user behavior

- previous examples

to make intelligent decisions or predictions.

------------------------------------------------------------------------

**How AI Works**

Basic AI workflow:

Data\
↓\
Training\
↓\
AI Model\
↓\
Predictions / Decisions

AI systems are trained using large amounts of data.

The system identifies patterns and learns rules automatically.

------------------------------------------------------------------------

**Main Types of AI**

**1. Narrow AI (Weak AI)**

Designed for a specific task.

Examples:

- recommendation systems

- chatbots

- voice assistants

Most AI today is Narrow AI.

**2. General AI (Strong AI)**

A theoretical AI that can perform any intellectual task like a human.

This does not fully exist yet.

------------------------------------------------------------------------

**Major Areas of AI**

**Machine Learning (ML)**

Machines learn from data without explicit programming.

Examples:

- fraud detection

- product recommendations

- price prediction

------------------------------------------------------------------------

**Deep Learning**

Advanced ML using neural networks.

Used for:

- image recognition

- speech recognition

- generative AI

------------------------------------------------------------------------

**Natural Language Processing (NLP)**

Helps machines understand human language.

Examples:

- ChatGPT

- translation apps

- sentiment analysis

**Computer Vision**

Allows machines to understand images and videos.

Examples:

- facial recognition

- medical imaging

- self-driving cars

------------------------------------------------------------------------

**AI vs Traditional Programming**

Traditional programming:

Rules + Data → Output

AI / Machine Learning:

Data + Output Examples → Learned Rules

Instead of manually coding rules, AI learns patterns automatically.
