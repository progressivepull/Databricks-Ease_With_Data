# EXPLANATION 04 9

**Question 9**

**Which portal is used to centrally manage Databricks workspaces after deployment?**

**Correct Answer:**\
✅ **account.azuredatabricks.net (Databricks Account Console)**

**Explanation**

The Databricks Account Console is used for **account-level management**.

From here, administrators can manage:

- Users

- Groups

- Service Principals

- Unity Catalog

- Workspaces

- Billing settings

- Account-wide features

The Azure portal is still used for Azure resources such as subscriptions and VNets, but the Account Console is where Databricks-specific administration occurs.
