# EXPLANATION 04 8

**Question 8**

**During networking configuration for Azure Databricks, which setting is recommended?**

**Correct Answer:**\
✅ **Disable public IP addresses and use the custom Virtual Network**

**Explanation**

Enterprise deployments typically follow a **private networking** model.

Recommended practices include:

- Disable public IP addresses for compute resources when supported by your deployment architecture.

- Use a customer-managed Virtual Network (VNet).

- Restrict internet exposure.

- Use private endpoints where appropriate.

This reduces the attack surface and improves security.
