# EXPLANATION 04 4

**Question 4**

**Azure Databricks billing consists of which two primary cost categories?**

**Correct Answer:**\
✅ **Databricks Units (DBUs) and Azure cloud resource charges**

**Explanation**

Azure Databricks billing consists of **two separate charges**.

**1. Databricks Units (DBUs)**

DBUs measure how much Databricks compute is consumed.

Examples include:

- Interactive clusters

- Job clusters

- SQL Warehouses

**2. Azure Infrastructure Charges**

Microsoft Azure bills separately for infrastructure such as:

- Virtual Machines

- Storage

- Networking

- Managed disks

Think of it as:

Total Cost\
\
=\
\
Databricks Service (DBUs)\
\
+\
\
Azure Infrastructure
