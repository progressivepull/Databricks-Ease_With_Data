# EXPLANATION 04 2

**Question 2**

**Which Azure subscription type is required to deploy Azure Databricks?**

**Correct Answer:**\
✅ **Pay-As-You-Go subscription**

**Explanation**

Azure Databricks creates Azure resources such as:

- Virtual Machines

- Managed Disks

- Networking

- Storage Accounts

- Load Balancers

These resources generate Azure charges.

A **Pay-As-You-Go** subscription is commonly used because it allows Azure to bill for these cloud resources as they are consumed. Enterprise subscriptions also support Azure Databricks, but Pay-As-You-Go is the standard subscription type commonly used for learning and many deployments.
