# EXPLANATION 04 3

**Question 3**

**Which permission level is required to create an Azure Databricks workspace?**

**Correct Answer:**\
✅ **Contributor**

**Explanation**

The Azure **Contributor** role allows a user to:

- Create resources

- Modify resources

- Delete resources

This is sufficient to deploy an Azure Databricks workspace.

The **Reader** role cannot create resources.

The **Owner** role also works but grants more permissions than necessary.

Microsoft recommends using the principle of least privilege whenever possible.
