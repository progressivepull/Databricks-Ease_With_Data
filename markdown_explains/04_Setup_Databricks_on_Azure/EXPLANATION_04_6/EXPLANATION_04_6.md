# EXPLANATION 04 6

**Question 6**

**What is the primary purpose of an Azure Resource Group during Azure Databricks deployment?**

**Correct Answer:**\
✅ **To organize and manage related Azure resources together**

**Explanation**

A Resource Group is simply a logical container.

It groups together related Azure resources such as:

- Azure Databricks Workspace

- Storage Account

- Virtual Network

- Network Security Groups

- Managed Identities

Benefits include:

- Easier management

- Easier deletion

- Simplified permissions

- Better organization

Think of it like a folder on your computer that contains everything related to one project.
