# EXPLANATION 04 1

**Question 1**

**Which cloud providers currently support Databricks deployments?**

**Correct Answer:**\
✅ **AWS, Microsoft Azure, Google Cloud Platform (GCP)**

**Explanation**

Databricks is a cloud-native platform and is officially available on **three major public cloud providers**:

| **Cloud Provider**    | **Databricks Product**     |
|-----------------------|----------------------------|
| Amazon Web Services   | Databricks on AWS          |
| Microsoft Azure       | Azure Databricks           |
| Google Cloud Platform | Databricks on Google Cloud |

Although the underlying cloud infrastructure differs, the Databricks experience is very similar across all three clouds.

Common features include:

- Unity Catalog

- Delta Lake

- Workflows

- SQL Warehouses

- MLflow

- Photon

- Notebooks

The primary differences are:

| **AWS**   | **Azure**              | **GCP**               |
|-----------|------------------------|-----------------------|
| Amazon S3 | ADLS Gen2              | Google Cloud Storage  |
| IAM       | Microsoft Entra ID     | Google IAM            |
| EC2       | Azure Virtual Machines | Google Compute Engine |

Databricks documentation is available separately for each supported cloud.
