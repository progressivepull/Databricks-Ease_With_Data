# EXPLANATION 44 8

**Question 8**

**Which of the following is NOT a supported notification destination for Databricks SQL Alerts?**

**Correct Answer:**\
**E. GitHub Issues**

**Why E is Correct**

Databricks supports notification destinations such as:

- Email

- Slack

- Microsoft Teams

- Webhooks (depending on configuration)

GitHub Issues is **not** a native notification destination for SQL Alerts. Workspace administrators configure available destinations.

**Why the other choices are wrong**

Email, Slack, and Microsoft Teams are supported notification channels.

**Memory Tip**

**Collaboration tools? Yes. GitHub Issues? No.**
