# EXPLANATION 44 5

**Question 5**

**Which comparison operator is supported when configuring a SQL Alert trigger condition?**

**Correct Answer:**\
**A. Greater than (\>)**

**Why A is Correct**

SQL Alerts support common comparison operators such as:

- 

- \<

- =

- \<=

- =

- !=

For example:

Pending Orders \> 10000

When the query result exceeds the threshold, the alert triggers.

**Why the other choices are wrong**

Unsupported or non-comparison operators cannot be used as alert threshold conditions.

**Memory Tip**

**Think SQL WHERE operators.**
