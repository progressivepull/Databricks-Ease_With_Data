# EXPLANATION 32 10

**Question 10**

**Which statement correctly compares Azure Key Vault-backed and Databricks-backed Secret Scopes?**

**Correct Answer:** **B. Azure Key Vault-backed scopes store secrets in Azure Key Vault, while Databricks-backed scopes store secrets inside Databricks.**

**Explanation**

| **Azure Key Vault-backed** | **Databricks-backed** |
|----|----|
| Secrets remain in Azure Key Vault | Secrets are stored in Databricks' encrypted secret store |
| Azure manages the secrets | Databricks manages the secrets |
| Commonly used when organizations already standardize on Azure Key Vault | Convenient for secrets managed entirely within Databricks |

**Why the Correct Answer is Correct**

The key distinction is **where the secret values are physically stored**.

**Why the Other Answers Are Wrong**

The incorrect options either reverse the storage locations or incorrectly state that both types store secrets in the same place.
