# EXPLANATION 32 5

**Question 5**

**Which DBUtils command displays all Secret Scopes that are accessible to the current user?**

**Correct Answer:** **C. dbutils.secrets.listScopes()**

**Explanation**

dbutils.secrets.listScopes() lists every Secret Scope the current user has permission to access.

**Why the Correct Answer is Correct**

It returns the available Secret Scope names without revealing the stored secret values.

**Why the Other Answers Are Wrong**

Other commands list secrets within a scope or retrieve individual secret values.
