# EXPLANATION 32 2

**Question 2**

**Which two types of Secret Scopes are supported by Databricks?**

**Correct Answer:** **B. Azure Key Vault-backed and Databricks-backed Secret Scopes**

**Explanation**

Databricks supports two Secret Scope backends:

- **Azure Key Vault-backed Secret Scopes**

- **Databricks-backed Secret Scopes**

**Why the Correct Answer is Correct**

These are the two officially supported methods for storing and managing secrets.

**Comparison**

**Azure Key Vault-backed**

- Secrets remain in Azure Key Vault.

- Databricks reads them securely.

**Databricks-backed**

- Secrets are stored inside Databricks' encrypted secret store.

**Why the Other Answers Are Wrong**

Other combinations include services that Databricks does not support as Secret Scope backends.
