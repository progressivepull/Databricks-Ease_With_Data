# EXPLANATION 32 8

**Question 8**

**Which command is used to authenticate the Databricks CLI with a workspace?**

**Correct Answer:** **C. databricks auth login**

**Explanation**

The current Databricks CLI uses:

databricks auth login

This command authenticates the CLI with a Databricks workspace so it can execute CLI commands.

**Why the Correct Answer is Correct**

It is the recommended authentication command for modern versions of the Databricks CLI.

**Why the Other Answers Are Wrong**

Older or incorrect commands either no longer apply or are not used for authentication.
