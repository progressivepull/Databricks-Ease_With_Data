# EXPLANATION 32 7

**Question 7**

**How are Databricks-backed Secret Scopes typically created and managed?**

**Correct Answer:** **C. Using the Databricks CLI**

**Explanation**

Although scopes can also be created through the workspace UI and APIs, Databricks documentation commonly demonstrates creating and managing Databricks-backed Secret Scopes with the Databricks CLI.

Example:

databricks secrets create-scope my-scope

**Why the Correct Answer is Correct**

The Databricks CLI is the standard automation and scripting tool for Secret Scope management.

**Why the Other Answers Are Wrong**

They refer to services that do not manage Databricks-backed Secret Scopes.
