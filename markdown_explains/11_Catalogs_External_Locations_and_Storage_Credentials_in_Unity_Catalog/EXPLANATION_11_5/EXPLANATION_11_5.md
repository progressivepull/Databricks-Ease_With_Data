# EXPLANATION 11 5

**Question 5**

**What is the purpose of the DESCRIBE CATALOG EXTENDED command?**

**Correct Answer:**\
✅ **To display catalog metadata such as owner, comments, and managed storage information**

**Explanation**

Example:

DESCRIBE CATALOG EXTENDED finance;

Returns information such as:

- Owner

- Comments

- Creation Date

- Managed Storage Location

- Catalog Properties

Think of it as asking:

**"Tell me everything about this catalog."**

This command is useful for administrators who need to inspect catalog configuration.

**Example**

Finance Catalog\
\
Owner\

\
