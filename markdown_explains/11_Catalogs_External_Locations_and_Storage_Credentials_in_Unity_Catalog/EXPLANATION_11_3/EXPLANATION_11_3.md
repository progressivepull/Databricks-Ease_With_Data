# EXPLANATION 11 3

**Question 3**

**Why might an organization assign different managed locations to different catalogs?**

**Correct Answer:**\
✅ **To organize data into separate storage locations for governance and isolation**

**Explanation**

Large organizations often separate business domains.

Example:

Finance Catalog\
\
↓\
\
finance-storage\
\
HR Catalog\
\
↓\
\
hr-storage\
\
Sales Catalog\
\
↓\
\
sales-storage

Benefits include:

- Data isolation

- Security

- Easier auditing

- Regulatory compliance

- Independent storage management

For example, finance data may need stricter retention policies than marketing data.

Using separate managed locations helps enforce these organizational boundaries.
