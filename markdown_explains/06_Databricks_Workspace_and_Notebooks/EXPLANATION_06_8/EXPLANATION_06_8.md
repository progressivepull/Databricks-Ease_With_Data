# EXPLANATION 06 8

**Question 8**

**What is the primary purpose of Markdown (Text) cells in a Databricks notebook?**

**Correct Answer:**\
✅ **Document and explain notebook content without executing code**

**Explanation**

Markdown cells are used for documentation.

Common uses include:

- Titles

- Headings

- Explanations

- Instructions

- Architecture diagrams

- Images

- Links

Example:

\# Customer ETL\
\
This notebook loads customer data,\
cleans it,\
and writes it into Delta Lake.

Markdown improves readability and helps others understand the notebook.

It is never executed as code.
