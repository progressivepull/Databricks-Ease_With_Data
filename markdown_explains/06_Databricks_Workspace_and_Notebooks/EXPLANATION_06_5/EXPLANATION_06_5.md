# EXPLANATION 06 5

**Question 5**

**Which configuration is recommended to reduce cloud costs when creating a compute cluster?**

**Correct Answer:**\
✅ **Enable Auto Termination**

**Explanation**

Auto Termination automatically shuts down an idle cluster after a specified amount of inactivity.

Example:

Developer finishes work\
\
↓\
\
Cluster Idle\
\
↓\
\
30 Minutes\
\
↓\
\
Cluster Stops

Benefits:

- Lower Azure costs

- Prevents forgotten clusters

- Reduces VM charges

Since clusters consume cloud resources while running, enabling Auto Termination is a simple way to reduce unnecessary costs.
