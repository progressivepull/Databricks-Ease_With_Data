# EXPLANATION 06 2

**Question 2**

**Which of the following items can users store in their personal workspace folders?**

**Correct Answer:**\
✅ **Notebooks, queries, dashboards, MLflow experiments, and files**

**Explanation**

Each user has a personal workspace folder.

Typical contents include:

- 📒 Notebooks

- 📊 SQL Queries

- 📈 Dashboards

- 🤖 MLflow Experiments

- 📁 Files

This allows developers to organize their own work before sharing it with others.

Example:

Users\
\
└── John\
├── ETL Notebook\
├── Sales Dashboard\
├── ML Experiment\
├── SQL Query\
└── test.csv

Personal folders help keep projects organized while supporting collaboration when needed.
