# EXPLANATION 06 7

**Question 7**

**Which feature allows multiple programming languages to be used within the same Databricks notebook?**

**Correct Answer:**\
✅ **Magic Commands (such as %sql and %sh)**

**Explanation**

Magic Commands switch languages within a notebook.

Examples:

\# Python

%sql\
SELECT \*\
FROM employees

%scala\
println("Hello")

%r\
summary(df)

This allows developers to use the best language for each task without creating multiple notebooks.
