# EXPLANATION 06 9

**Question 9**

**Which Databricks notebook feature allows developers to restore previous versions of a notebook?**

**Correct Answer:**\
✅ **Version History**

**Explanation**

Version History automatically tracks notebook changes.

Developers can:

- View earlier versions

- Compare versions

- Restore previous versions

- Recover mistakes

Example:

Version 1\
\
↓\
\
Version 2\
\
↓\
\
Version 3\
\
↓\
\
Restore Version 1

This provides a safety net when experimenting with notebook changes.
