# EXPLANATION 22 1

**Databricks Workflows Explained (Questions 1–10)**

Databricks **Workflows** is the platform's orchestration service for automating data pipelines, machine learning workflows, and scheduled notebook execution. A workflow can run notebooks, SQL queries, Python scripts, Delta Live Tables (DLT) pipelines, JARs, and other task types in a defined sequence with dependencies.
