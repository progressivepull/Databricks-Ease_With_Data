# EXPLANATION 22 4

**Question 3**

**Which of the following is a supported Databricks Workflow task type?**

**Correct Answer: C. Delta Live Tables (DLT)**

**Explanation**

Databricks Workflows supports many task types, including:

- Notebook

- SQL

- Python script

- JAR

- Spark Submit

- dbt

- Delta Live Tables (DLT)

A **Delta Live Tables (DLT)** task runs declarative ETL pipelines that automatically manage:

- Data quality

- Dependencies

- Incremental processing

- Pipeline execution

DLT is commonly used for building reliable production data pipelines.
