# EXPLANATION 05 3

**Question 3**

**What additional tab becomes available after Unity Catalog is enabled?**

**Correct Answer:**\
✅ **Permissions**

**Explanation**

Once Unity Catalog is enabled, Databricks adds a **Permissions** tab because Unity Catalog introduces centralized governance.

From this tab you can manage permissions on:

- Catalogs

- Schemas

- Tables

- Views

- Volumes

- Functions

Instead of configuring permissions separately in every workspace, Unity Catalog lets you define them centrally and apply them consistently.

**Memory Trick**

Unity Catalog\
\
↓\
\
Permissions Tab
