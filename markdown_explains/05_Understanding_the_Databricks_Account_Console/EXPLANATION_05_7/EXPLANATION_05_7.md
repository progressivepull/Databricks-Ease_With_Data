# EXPLANATION 05 7

**Question 7**

**Why are Groups used in the Databricks Account Console?**

**Correct Answer:**\
✅ **To simplify permission management by assigning permissions to collections of users**

**Explanation**

Instead of assigning permissions individually:

John\
\
Mary\
\
Bob\
\
Susan

Create a group:

Data Engineers

Assign permissions once:

Finance Catalog\
\
↓\
\
Data Engineers Group

Every member automatically receives access.

Benefits:

- Easier administration

- Consistent permissions

- Faster onboarding

- Reduced errors

This is a standard role-based access control (RBAC) practice.
