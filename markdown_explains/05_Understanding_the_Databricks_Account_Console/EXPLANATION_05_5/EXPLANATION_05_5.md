# EXPLANATION 05 5

**Question 5**

**Who has permission to manage users through the Databricks Account Console?**

**Correct Answer:**\
✅ **Account Administrators**

**Explanation**

Only **Account Administrators** have account-wide authority.

They can:

- Add users

- Remove users

- Create groups

- Create Service Principals

- Create workspaces

- Enable Unity Catalog

- Configure account settings

Workspace Administrators manage users **inside a workspace**, but Account Administrators manage identities across the entire account.

**Memory Trick**

Workspace Admin\
\
↓\
\
One Workspace\
\
Account Admin\
\
↓\
\
Entire Account
