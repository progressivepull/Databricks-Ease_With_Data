# EXPLANATION 26 8

**Question 8**

**Which Delta Live Tables edition includes both Change Data Capture (CDC) and Data Quality Expectations?**

**Correct Answer:** **E. Advanced**

**Explanation**

The **Advanced** edition provides enterprise capabilities such as:

- CDC

- Data Quality Expectations

- Enhanced pipeline features

**Why the Correct Answer is Correct**

These advanced features are intended for production-grade ETL pipelines.

**Why the Other Answers Are Wrong**

Lower editions do not include the full set of enterprise capabilities.
