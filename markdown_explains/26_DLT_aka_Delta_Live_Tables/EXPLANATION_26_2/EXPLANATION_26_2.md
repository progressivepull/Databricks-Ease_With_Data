# EXPLANATION 26 2

**Question 2**

**Which operational tasks is automatically managed by Delta Live Tables?**

**Correct Answer:** **C. Pipeline orchestration, cluster management, and dependency tracking**

**Explanation**

DLT automatically determines:

- Pipeline execution order

- Required compute

- Dataset dependencies

- Monitoring and retries

- Pipeline lifecycle

**Why the Correct Answer is Correct**

Instead of manually creating Spark jobs and scheduling execution, DLT builds and executes the dependency graph automatically.

**Why the Other Answers Are Wrong**

Other options usually describe tasks developers still perform, such as writing transformation logic or designing schemas.
