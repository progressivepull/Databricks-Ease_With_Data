# EXPLANATION 26 3

**Question 3**

**Which DLT dataset type is designed for continuously arriving data and incremental processing?**

**Correct Answer:** **D. Streaming Table**

**Explanation**

Streaming Tables continuously process newly arriving data instead of rereading the entire dataset.

**Why the Correct Answer is Correct**

Streaming Tables are built on **Spark Structured Streaming** and automatically process incremental updates as new records arrive.

**Why the Other Answers Are Wrong**

- Materialized Views process batch-style computations.

- Views are temporary logical datasets.

- Standard tables are not designed for continuous ingestion.
