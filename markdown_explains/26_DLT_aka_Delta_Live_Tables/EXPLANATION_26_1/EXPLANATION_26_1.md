# EXPLANATION 26 1

**Question 1**

**What is the primary purpose of Delta Live Tables (DLT) in Databricks?**

**Correct Answer:** **B. To simplify ETL development by automatically managing pipeline operations while developers define transformations**

**Explanation**

Delta Live Tables (DLT) is a **declarative ETL framework**. Instead of writing code to manage clusters, job dependencies, retries, scheduling, and monitoring, developers simply define **what transformations should occur**. Databricks automatically handles the operational aspects of the pipeline. (Note: DLT has since been renamed **Lakeflow Spark Declarative Pipelines**, but the concepts remain the same.)

**Why the Correct Answer is Correct**

- Developers focus on business logic.

- Databricks manages orchestration, reliability, retries, monitoring, and optimization.

- Less infrastructure code and easier pipeline maintenance.

**Why the Other Answers Would Be Wrong**

- They describe infrastructure management or unrelated features rather than DLT's primary goal.
