# EXPLANATION 34 3

**Question 3**

**What happens when a privilege is granted on a parent object, such as a Catalog?**

**Correct Answer:** **B. Child objects automatically inherit the permission.**

**Explanation**

Unity Catalog supports a hierarchical permission model. When privileges are granted at a higher level (such as a Catalog or Schema), those permissions are inherited by child objects unless explicitly overridden.

**Why the Correct Answer is Correct**

Inheritance reduces administrative effort because permissions do not need to be granted individually to every table or view.

**Why the Other Answers Are Wrong**

The other options incorrectly state that permissions must always be assigned individually or that inheritance does not exist.
