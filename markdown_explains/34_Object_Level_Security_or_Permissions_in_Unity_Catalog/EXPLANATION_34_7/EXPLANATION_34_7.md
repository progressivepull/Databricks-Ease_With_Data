# EXPLANATION 34 7

**Question 7**

**Which privilege allows a user to read files stored in a Unity Catalog Volume?**

**Correct Answer:** **C. READ VOLUME**

**Explanation**

Unity Catalog Volumes store unstructured data such as images, documents, and machine learning artifacts. The **READ VOLUME** privilege allows users to read files stored within a volume.

**Why the Correct Answer is Correct**

READ VOLUME is specifically designed for file access within Unity Catalog Volumes.

**Why the Other Answers Are Wrong**

Other privileges apply to tables or schemas rather than volume files.
