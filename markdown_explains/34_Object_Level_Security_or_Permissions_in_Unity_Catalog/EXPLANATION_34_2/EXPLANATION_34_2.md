# EXPLANATION 34 2

**Question 2**

**Which of the following correctly represents the Unity Catalog object hierarchy?**

**Correct Answer:** **B. Metastore → Catalog → Schema → Tables, Views, Volumes, Functions, and Models**

**Explanation**

Unity Catalog organizes data assets in a hierarchical structure:

Metastore

└── Catalog

└── Schema

├── Tables

├── Views

├── Volumes

├── Functions

└── Models

This hierarchy simplifies governance by allowing permissions to be assigned at different levels.

**Why the Correct Answer is Correct**

Every Unity Catalog object belongs somewhere within this hierarchy, enabling inheritance of permissions.

**Why the Other Answers Are Wrong**

The incorrect options either reverse the hierarchy or omit required levels.
