# EXPLANATION 34 1

**Question 1**

**What is the primary purpose of Object-Level Security in Unity Catalog?**

**Correct Answer:** **C. To control who can view, access, modify, and manage individual Unity Catalog objects**

**Explanation**

Object-Level Security in Unity Catalog provides fine-grained access control over data assets. Administrators can specify who can view, query, modify, or manage individual objects such as catalogs, schemas, tables, views, volumes, functions, and machine learning models. This enables organizations to enforce the principle of least privilege and meet security and compliance requirements.

**Why the Correct Answer is Correct**

Object-Level Security allows administrators to:

- Control access to individual Unity Catalog objects.

- Grant different permissions to different users or groups.

- Protect sensitive data while allowing appropriate collaboration.

- Enforce governance across the entire data platform.

**Why the Other Answers Are Wrong**

- They either focus only on authentication instead of authorization.

- They limit security to clusters or workspaces rather than data objects.

- They do not provide fine-grained control over individual assets.
