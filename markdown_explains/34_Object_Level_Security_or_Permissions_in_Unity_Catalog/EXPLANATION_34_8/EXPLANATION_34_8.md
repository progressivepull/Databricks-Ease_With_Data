# EXPLANATION 34 8

**Question 8**

**Which statement about ALL PRIVILEGES is correct?**

**Correct Answer:** **C. It grants nearly every operational permission but does not include managing permissions or ownership.**

**Explanation**

The **ALL PRIVILEGES** grant provides most operational capabilities, such as reading and modifying data where appropriate. However, it does **not** automatically include ownership or permission-management capabilities like MANAGE or object ownership. Those privileges must be granted separately.

**Why the Correct Answer is Correct**

This prevents users from unintentionally gaining administrative control over objects.

**Why the Other Answers Are Wrong**

The incorrect options assume ALL PRIVILEGES grants complete administrative ownership, which it does not.
