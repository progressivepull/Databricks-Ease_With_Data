# EXPLANATION 34 6

**Question 6**

**What does the BROWSE privilege allow a user to do?**

**Correct Answer:** **C. Discover catalogs, view metadata, and see lineage without accessing the data itself**

**Explanation**

The **BROWSE** privilege allows users to discover Unity Catalog objects, inspect metadata, and view lineage information without granting access to the underlying data. This helps users find relevant datasets while maintaining security.

**Why the Correct Answer is Correct**

BROWSE improves discoverability without exposing sensitive table contents.

**Why the Other Answers Are Wrong**

The incorrect options confuse metadata access with actual data access.
