# EXPLANATION 45 9

**Question 9**

**What advantage do AI/BI Dashboards gain by using Metric Views?**

**Correct Answer:**\
**A. Dashboard developers no longer need to define complex SQL calculations for every report because reusable business logic already exists.**

**Why A is Correct**

Metric Views separate business logic from dashboards.

Instead of each dashboard containing:

Revenue SQL

Profit SQL

Margin SQL

the dashboard simply references predefined measures.

Benefits include:

- simpler dashboards

- faster development

- consistent KPIs

- easier maintenance

**Why the other choices are wrong**

Metric Views do not replace dashboards; they simplify and standardize the logic behind them.

**Memory Tip**

**Dashboard = Visual**

**Metric View = Business Logic**
