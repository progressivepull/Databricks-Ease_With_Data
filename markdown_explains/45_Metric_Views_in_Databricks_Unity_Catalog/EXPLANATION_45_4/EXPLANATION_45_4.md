# EXPLANATION 45 4

**Question 4**

**What is the primary benefit of defining business logic once in a Metric View?**

**Correct Answer:**\
**B. It ensures everyone uses the same KPI calculations and reduces duplicated SQL.**

**Why B is Correct**

Without Metric Views:

Dashboard A

Revenue SQL

Dashboard B

Different Revenue SQL

Dashboard C

Another Revenue SQL

Results become inconsistent.

With Metric Views:

Revenue

↓

One Definition

↓

Everyone Uses It

Benefits include:

- consistent KPIs

- less duplicated SQL

- easier maintenance

- trusted analytics

**Memory Tip**

**Write Once. Use Everywhere.**
