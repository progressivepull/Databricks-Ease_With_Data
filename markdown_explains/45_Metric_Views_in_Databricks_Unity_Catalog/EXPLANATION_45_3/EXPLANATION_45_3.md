# EXPLANATION 45 3

**Question 3**

**How are Metric Views created in Databricks?**

**Correct Answer:**\
**C. Using YAML definitions**

**Why C is Correct**

Metric Views are defined using **YAML**, which describes:

- source tables

- joins

- dimensions

- measures

- filters

- metadata

YAML separates business definitions from SQL queries, making the semantic model easier to maintain.

**Why the other choices are wrong**

Metric Views are not primarily authored as:

- notebooks

- Python classes

- JSON documents
