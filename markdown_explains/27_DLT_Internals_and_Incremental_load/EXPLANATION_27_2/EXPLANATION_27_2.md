# EXPLANATION 27 2

**Question 2**

**What is created each time a Delta Live Tables pipeline executes?**

**Correct Answer:** **C. An Update containing execution status, logs, metrics, and Spark UI information.**

**Explanation**

Every execution of a DLT pipeline is called an **Update**. Each update records:

- Execution status

- Pipeline logs

- Processing metrics

- Spark UI

- Errors

- Duration

- Progress information

**Why the Correct Answer is Correct**

Updates provide complete observability into each pipeline execution, making it easier to debug failures and monitor performance.

**Why the Other Answers Are Wrong**

The other options describe objects or metadata that are not generated for every execution.
