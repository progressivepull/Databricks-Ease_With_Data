# EXPLANATION 27 10

**Question 10**

**Which statement about Unity Catalog Lineage is correct?**

**Correct Answer:** **C. It provides end-to-end table and column-level lineage for DLT pipelines and other Unity Catalog-managed assets.**

**Explanation**

Unity Catalog automatically captures lineage information, showing:

- Upstream and downstream datasets

- Table-level lineage

- Column-level lineage

- Pipeline relationships

- Data flow across the Lakehouse

**Why the Correct Answer is Correct**

Lineage helps users understand where data originated, how it was transformed, and which downstream assets depend on it.

**Why the Other Answers Are Wrong**

The other options incorrectly limit lineage to only tables, only notebooks, or only DLT pipelines.
