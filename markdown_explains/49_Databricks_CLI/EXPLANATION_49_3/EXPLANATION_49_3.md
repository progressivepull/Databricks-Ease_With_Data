# EXPLANATION 49 3

**Question 3**

**Which of the following is a key benefit of using the Databricks CLI?**

**Correct Answer:**\
**B. It enables automation, scripting, and CI/CD workflows.**

**Why B is Correct**

Because the CLI can be called from:

- Bash

- PowerShell

- Python scripts

- Azure Pipelines

- GitHub Actions

- Jenkins

- GitLab CI

it becomes an excellent automation tool.

Typical automated tasks include:

- Deploying notebooks

- Deploying jobs

- Deploying DABs

- Managing Unity Catalog

- Running tests

([docs.databricks.com](https://docs.databricks.com/aws/en/dev-tools/cli/))

**Why the other choices are wrong**

The CLI is not primarily a reporting tool or dashboard builder.
