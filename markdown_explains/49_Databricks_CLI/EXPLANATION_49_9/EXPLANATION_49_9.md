# EXPLANATION 49 9

**Question 9**

**Which command displays help information for Databricks CLI commands and subcommands?**

**Correct Answer:**\
**C. databricks --help**

**Why C is Correct**

The help system displays:

- Available commands

- Command syntax

- Subcommands

- Options

- Examples

You can also request help for a specific command:

databricks jobs --help

This is the quickest way to explore the CLI interactively. ([docs.databricks.com](https://docs.databricks.com/aws/en/dev-tools/cli/))

**Why the other choices are wrong**

Version commands display version information, not command documentation.
