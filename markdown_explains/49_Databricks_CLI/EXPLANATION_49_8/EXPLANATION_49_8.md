# EXPLANATION 49 8

**Question 8**

**Which command lists all Unity Catalog catalogs available to the authenticated user?**

**Correct Answer:**\
**C. databricks catalogs list**

**Why C is Correct**

The command:

databricks catalogs list

returns the Unity Catalog catalogs that the authenticated user is permitted to access.

This command is commonly used to verify permissions and confirm that authentication is working correctly. ([docs.databricks.com](https://docs.databricks.com/aws/en/dev-tools/cli/reference/catalogs-commands))

**Why the other choices are wrong**

Other commands manage different resources such as schemas, jobs, or workspaces.
