# ANSWER 35 7

**Question 7**

**Which keyword is used when defining a Python User-Defined Function in Unity Catalog?**

**A. LANGUAGE SCALA ❌ Incorrect**

Defines a Scala function.

------------------------------------------------------------------------

**B. LANGUAGE JAVA ❌ Incorrect**

Not used for Python.

------------------------------------------------------------------------

**C. LANGUAGE PYTHON ✅ Correct**

Example:

CREATE FUNCTION tax(...)

LANGUAGE PYTHON

AS ...

This tells Databricks the function body contains Python code.

------------------------------------------------------------------------

**D. PYTHON FUNCTION ❌ Incorrect**

Not valid SQL syntax.

------------------------------------------------------------------------

**E. CREATE PYTHON FUNCTION ❌ Incorrect**

Also invalid syntax.

------------------------------------------------------------------------

**Why C is correct**

LANGUAGE PYTHON explicitly declares that the function implementation is written in Python.

------------------------------------------------------------------------
