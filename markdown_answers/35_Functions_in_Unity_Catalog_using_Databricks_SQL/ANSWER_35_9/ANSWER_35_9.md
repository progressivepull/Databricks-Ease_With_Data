# ANSWER 35 9

**Question 9**

**Which information can be viewed for a registered function in Catalog Explorer?**

**A. Only the function name and owner ❌ Incorrect**

Catalog Explorer shows much more.

------------------------------------------------------------------------

**B. Only execution history ❌ Incorrect**

Execution history is not the primary information displayed.

------------------------------------------------------------------------

**C. Input parameters, return type, function type, language, and definition ✅ Correct**

Catalog Explorer displays metadata including:

- function name

- owner

- parameters

- return type

- scalar/table function

- SQL or Python

- definition

- permissions

This helps developers understand and reuse functions.

------------------------------------------------------------------------

**D. Only the underlying Delta table location ❌ Incorrect**

Functions are not Delta tables.

------------------------------------------------------------------------

**E. Only the Python source code ❌ Incorrect**

SQL functions are also supported.

------------------------------------------------------------------------

**Why C is correct**

Catalog Explorer provides rich metadata about registered functions, making them easier to discover, understand, and govern.

------------------------------------------------------------------------
