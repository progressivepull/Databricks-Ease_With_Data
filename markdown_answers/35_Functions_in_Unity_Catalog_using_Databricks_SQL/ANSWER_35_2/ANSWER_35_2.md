# ANSWER 35 2

**Question 2**

**Which of the following is an example of a built-in Databricks SQL function?**

**A. tax_sql() ❌ Incorrect**

Looks like a custom function.

Not built into Databricks SQL.

------------------------------------------------------------------------

**B. get_emp() ❌ Incorrect**

Also appears to be user-defined.

------------------------------------------------------------------------

**C. LOWER() ✅ Correct**

LOWER() is a standard built-in SQL function.

Example:

SELECT LOWER('Databricks');

Result:

databricks

Other built-in functions include:

- UPPER()

- LENGTH()

- ROUND()

- ABS()

- SUBSTRING()

- CONCAT()

**D. nutrition_api() ❌ Incorrect**

Appears to be a custom function.

------------------------------------------------------------------------

**E. employee_lookup() ❌ Incorrect**

Also resembles a user-created function.

------------------------------------------------------------------------

**Why C is correct**

LOWER() is part of the built-in SQL function library available in Databricks.

------------------------------------------------------------------------
