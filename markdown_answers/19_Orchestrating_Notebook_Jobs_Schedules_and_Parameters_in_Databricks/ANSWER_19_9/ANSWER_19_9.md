# ANSWER 19 9

**Question 9**

**Which of the following can be configured when scheduling a Databricks notebook as a recurring job?**

**A. Only the notebook name** ❌ Incorrect

- Jobs have many configurable options.

**B. Only the cluster type** ❌ Incorrect

- Cluster choice is only one setting.

**C. Frequency, widget parameters, notifications, time zone, and Cron expressions** ✅ Correct

Databricks Jobs allow configuration of:

- Schedule

- Cron

- Time Zone

- Parameters

- Email notifications

- Failure alerts

- Retry policy

- Cluster

- Notebook

This makes jobs suitable for production automation.

**D. Only Spark configuration settings** ❌ Incorrect

- Spark settings are only a small part.

**E. Only Unity Catalog permissions** ❌ Incorrect

- Unity Catalog permissions are unrelated to scheduling.
