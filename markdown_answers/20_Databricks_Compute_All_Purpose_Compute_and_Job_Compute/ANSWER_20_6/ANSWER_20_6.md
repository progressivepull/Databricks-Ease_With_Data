# ANSWER 20 6

**Question 6**

**What is the recommendation when selecting a Databricks Runtime (DBR) version?**

**A. Always use the oldest supported version.** ❌ Incorrect

- Older versions miss newer features and fixes.

**B. Use the latest Long-Term Support (LTS) version whenever possible.** ✅ Correct

LTS releases provide:

- Stability

- Bug fixes

- Security updates

- Long support period

They're recommended for production workloads.

**C. Use only ML runtimes.** ❌ Incorrect

- ML runtimes are only for machine learning workloads.

**D. Use Photon regardless of runtime version.** ❌ Incorrect

- Photon is recommended when supported, but selecting an appropriate runtime version comes first.

**E. Always select the preview runtime.** ❌ Incorrect

- Preview releases are for testing.

- Not recommended for production.

Latest LTS
