# ANSWER 20 5

**Question 5**

**Which Access Mode supports multiple users while remaining compatible with Unity Catalog?**

**A. Single User only** ❌ Incorrect

- Single User mode allows only one user.

**B. No Isolation Shared** ❌ Incorrect

- This is a legacy access mode.

- It is **not recommended** for Unity Catalog.

**C. Shared** ✅ Correct

Shared Access Mode allows:

- Multiple users

- Unity Catalog compatibility

- Secure isolation

- Fine-grained permissions

Ideal for collaborative environments.

**D. Legacy Shared** ❌ Incorrect

- Legacy Shared predates Unity Catalog.

- Shared Access Mode is the recommended replacement.

**E. Driver Only** ❌ Incorrect

- Driver Only is not an access mode.

**Memory Trick**

Many Users

↓

Shared
