# ANSWER 20 3

**Question 3**

**Which type of compute is best suited for interactive notebook development and manual execution?**

**A. Job Compute** ❌ Incorrect

- Job Compute is intended for automated jobs.

- It starts and stops automatically.

**B. Serverless Compute** ❌ Incorrect

- Serverless is excellent for many workloads but the lesson specifically focuses on traditional compute types.

- The recommended answer for interactive notebook development is All-Purpose Compute.

**C. All-Purpose Compute** ✅ Correct

All-Purpose Compute is ideal for:

- Writing notebooks

- Interactive development

- Testing code

- Data exploration

- Debugging

It stays running until stopped or auto-terminated.

**D. SQL Warehouse** ❌ Incorrect

- SQL Warehouses are optimized for SQL analytics.

**E. Delta Live Compute** ❌ Incorrect

- Delta Live Tables use compute for pipeline execution, not interactive notebook development.
