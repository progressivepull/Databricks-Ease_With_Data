# ANSWER 20 7

**Question 7**

**What is the primary benefit of enabling Photon?**

**A. It reduces the number of worker nodes to zero.** ❌ Incorrect

- Photon doesn't eliminate workers.

**B. It automatically creates Unity Catalog objects.** ❌ Incorrect

- Photon has nothing to do with Unity Catalog.

**C. It improves ETL, SQL, streaming, and data science performance through a high-performance execution engine.** ✅ Correct

Photon is a high-performance execution engine.

Benefits:

- Faster SQL

- Faster ETL

- Faster joins

- Better aggregation performance

- Improved DataFrame operations

It accelerates many workloads without requiring code changes.

**D. It eliminates the need for Spark.** ❌ Incorrect

- Photon works **with** Spark.

**E. It disables Auto Scaling.** ❌ Incorrect

- Photon and Auto Scaling are independent features.
