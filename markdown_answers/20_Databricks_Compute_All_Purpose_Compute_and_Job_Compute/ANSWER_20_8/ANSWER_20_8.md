# ANSWER 20 8

**Question 8**

**What does Auto Scaling do for a Databricks cluster?**

**A. Automatically upgrades the Databricks Runtime version.** ❌ Incorrect

- Runtime upgrades are manual.

**B. Automatically adjusts the number of worker nodes based on workload demand.** ✅ Correct

Example:

Morning:

2 Workers

Heavy workload:

8 Workers

Workload finishes:

2 Workers

Benefits:

- Better performance

- Lower costs

- No manual resizing

**C. Automatically converts All-Purpose Compute into Job Compute.** ❌ Incorrect

- Compute types never convert automatically.

**D. Automatically enables Photon.** ❌ Incorrect

- Photon must be selected separately.

**E. Automatically creates notebooks.** ❌ Incorrect

- Auto Scaling only affects worker count.

**Memory Trick**

Auto Scaling

↓

Workers Go Up

Workers Go Down
