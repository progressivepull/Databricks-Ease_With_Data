# ANSWER 05 3

**Question 3**

**What additional tab becomes available after Unity Catalog is enabled?**

**A. Compute** ❌ Incorrect

- Compute exists regardless of Unity Catalog.

- It manages clusters and SQL Warehouses.

**B. Storage** ❌ Incorrect

- Storage configuration is separate from Unity Catalog permissions.

**C. Permissions** ✅ **Correct**

- Once Unity Catalog is enabled, administrators gain access to a **Permissions** tab.

- This allows management of:

  - Catalog permissions

  - Schema permissions

  - Table permissions

  - Volume permissions

  - External location permissions

**D. Monitoring** ❌ Incorrect

- Monitoring is unrelated to enabling Unity Catalog.

**E. Jobs** ❌ Incorrect

- Jobs schedule notebook and workflow execution.

- They are independent of Unity Catalog.

**Key Concept**

Unity Catalog introduces centralized governance, making the **Permissions** tab essential for controlling access to data.
