# ANSWER 05 2

**Question 2**

**Which section of the Account Console allows administrators to view and configure individual Databricks workspaces?**

**A. Catalog** ❌ Incorrect

- Catalog is used for Unity Catalog governance.

- It manages data assets—not workspaces.

**B. Workspaces** ✅ **Correct**

- The Workspaces page lists every workspace in the Databricks account.

- Administrators can:

  - View workspaces

  - Configure workspace settings

  - Assign metastores

  - Manage workspace information

**C. Previews** ❌ Incorrect

- Previews contains experimental or upcoming Databricks features.

- It is unrelated to workspace management.

**D. Settings** ❌ Incorrect

- Settings manages account-wide features.

- It does not list or configure individual workspaces.

**E. Groups** ❌ Incorrect

- Groups manages collections of users.

- It is not used to manage workspaces.

**Key Concept**

Every Databricks deployment can contain **multiple workspaces**, all managed centrally from the **Workspaces** section.
