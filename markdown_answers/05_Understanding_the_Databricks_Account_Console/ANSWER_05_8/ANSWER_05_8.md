# ANSWER 05 8

**Question 8**

**What is the purpose of Network Connectivity Configurations (NCC)?**

**A. To optimize SQL query performance** ❌ Incorrect

- NCC is about networking, not query optimization.

**B. To securely connect Azure resources that cannot communicate directly with Databricks** ✅ **Correct**

- NCC securely connects Databricks with customer-managed cloud resources.

- It is used when private or controlled network communication is required.

**C. To create additional Spark clusters** ❌ Incorrect

- Clusters are created from Compute.

**D. To manage notebook permissions** ❌ Incorrect

- Notebook permissions are handled through access controls and Unity Catalog where applicable.

**E. To configure Auto Loader** ❌ Incorrect

- Auto Loader ingests data files.

- It does not configure networking.

**Key Concept**

NCC is about **secure private networking**, not analytics or compute.
