# ANSWER 10 4

**Question 4**

**Which Azure feature must be enabled to use Azure Data Lake Storage Gen2 (ADLS Gen2)?**

**A. Soft Delete** ❌ Incorrect

- Soft Delete protects deleted files.

- It is optional.

**B. Geo-Redundant Storage (GRS)** ❌ Incorrect

- GRS replicates data across regions.

- It is unrelated to ADLS Gen2 functionality.

**C. Hierarchical Namespace** ✅ Correct

Hierarchical Namespace (HNS) transforms Blob Storage into Data Lake Storage Gen2.

It enables:

- Directories

- Folder structures

- File-level operations

- Better analytics performance

Without HNS:

Blob Storage

With HNS:

ADLS Gen2

**D. Customer-Managed Keys** ❌ Incorrect

- CMKs provide encryption control.

- They are optional.

**E. Private Endpoints** ❌ Incorrect

- Private Endpoints improve network security.

- They are not required for ADLS Gen2.

**Memory Trick**

No HNS → Not ADLS Gen2
