# ANSWER 32 8

**Question 8**

**Which command is used to authenticate the Databricks CLI with a workspace?**

**A. databricks login ❌ Incorrect**

- Not the current CLI authentication command.

------------------------------------------------------------------------

**B. databricks connect ❌ Incorrect**

- connect is related to Databricks Connect, not CLI authentication.

------------------------------------------------------------------------

**C. databricks auth login ✅ Correct**

Example:

databricks auth login

This authenticates the CLI so it can communicate securely with a Databricks workspace.

------------------------------------------------------------------------

**D. databricks workspace login ❌ Incorrect**

- Not a valid CLI command.

------------------------------------------------------------------------

**E. databricks authenticate ❌ Incorrect**

- Not a supported command.

------------------------------------------------------------------------

**Why C is correct**

databricks auth login is the recommended authentication command for the modern Databricks CLI.

------------------------------------------------------------------------
