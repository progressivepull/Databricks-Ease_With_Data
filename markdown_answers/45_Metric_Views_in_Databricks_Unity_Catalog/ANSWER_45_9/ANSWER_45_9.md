# ANSWER 45 9

**Question 9**

**What advantage do AI/BI Dashboards gain by using Metric Views?**

**A. Dashboard developers no longer need to define complex SQL calculations for every report because reusable business logic already exists.** ✅ Correct

Without Metric Views

Every dashboard writes:

SUM(...)

COUNT(...)

AVG(...)

CASE...

With Metric Views

Simply write:

measure(total_sales)

Everything is already defined.

Benefits:

- Faster dashboard creation

- Consistent KPIs

- Less SQL

- Easier maintenance

**B. Dashboards can only display KPI cards.** ❌ Incorrect

- Dashboards support many visualization types.

**C. SQL Warehouses are no longer required.** ❌ Incorrect

- SQL Warehouses still execute queries.

**D. All dashboards become Streaming Tables automatically.** ❌ Incorrect

- Completely unrelated.

**E. Forecasting requires manual SQL coding for every visualization.** ❌ Incorrect

- Metric Views reduce manual SQL.

**Key Concept**

**Metric Views make dashboards much easier to build.**
