# ANSWER 08 7

**Question 7**

**What types of data can Unity Catalog Volumes store?**

**A. Only structured relational tables** ❌ Incorrect

- Tables are stored differently from volumes.

**B. Only machine learning models** ❌ Incorrect

- ML models can be stored, but volumes are not limited to them.

**C. Structured, semi-structured, and unstructured data** ✅ Correct

Volumes can store nearly any file.

Examples:

- PDFs

- Images

- Videos

- CSV

- JSON

- XML

- ZIP

- Text files

- Machine learning artifacts

**D. Only JSON and CSV files** ❌ Incorrect

- Much too limited.

**E. Only Delta transaction logs** ❌ Incorrect

- Delta logs are separate.

**Memory trick**

Volume = **Folder of files**
