# ANSWER 08 8

**Question 8**

**Which of the following is an example of a non-data securable object managed by Unity Catalog?**

**A. Table** ❌ Incorrect

- Tables contain data.

**B. View** ❌ Incorrect

- Views expose data.

**C. Volume** ❌ Incorrect

- Volumes contain files.

**D. Storage Credential** ✅ Correct

Storage Credentials store authentication information used to access cloud storage securely.

Examples:

- Azure Managed Identity

- AWS IAM Role

- GCP Service Account

They are governance objects rather than data objects.

**E. Schema** ❌ Incorrect

- Schemas organize data.

**Examples of non-data securables**

- Storage Credentials

- External Locations

- Connections

- Credentials
