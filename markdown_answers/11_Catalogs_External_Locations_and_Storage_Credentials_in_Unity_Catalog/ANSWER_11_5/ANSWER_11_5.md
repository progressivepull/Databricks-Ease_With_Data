# ANSWER 11 5

**Question 5**

**What is the purpose of the DESCRIBE CATALOG EXTENDED command?**

**A. To create a catalog** ❌ Incorrect

- Creating uses:

CREATE CATALOG

**B. To delete a catalog** ❌ Incorrect

- Deleting uses:

DROP CATALOG

**C. To display catalog metadata such as owner, comments, and managed storage information** ✅ Correct

This command displays information including:

- Owner

- Comment

- Managed Location

- Creation Date

- Properties

Example:

DESCRIBE CATALOG EXTENDED sales;

Think of it as asking:

"Tell me everything about this catalog."

**D. To optimize Delta tables** ❌ Incorrect

- Optimization uses:

OPTIMIZE

**E. To assign workspace permissions** ❌ Incorrect

- Permissions are assigned with GRANT.

**Memory Trick**

DESCRIBE

↓

Read information

Never changes anything.
