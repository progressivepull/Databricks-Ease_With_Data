# ANSWER 40 2

**Question 2**

**Which task is still the responsibility of a user when using Serverless Compute?**

**A. Selecting virtual machine sizes** ❌ Incorrect

- Databricks chooses the VM sizes automatically.

**B. Configuring autoscaling policies** ❌ Incorrect

- Autoscaling happens automatically.

- Users don't configure scaling rules.

**C. Applying Serverless to the workload and running the code** ✅ Correct

- Users still decide:

  - Which notebook or workflow to run

  - Whether to use Serverless Compute

  - Execute their code

- Databricks manages the infrastructure afterward.

**D. Maintaining Spark clusters** ❌ Incorrect

- Serverless eliminates cluster maintenance.

**E. Patching operating systems on worker nodes** ❌ Incorrect

- Databricks manages OS updates.

**Key Concept**

Users manage workloads. Databricks manages infrastructure.
