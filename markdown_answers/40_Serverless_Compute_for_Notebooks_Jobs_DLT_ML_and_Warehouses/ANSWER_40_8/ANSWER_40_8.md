# ANSWER 40 8

**Question 8**

**Which Databricks workloads can use Serverless Compute?**

**A. Only notebooks** ❌ Incorrect

- Serverless supports much more.

**B. Only SQL Warehouses** ❌ Incorrect

- SQL Warehouses are only one workload.

**C. Notebooks, Workflows (Jobs), Delta Live Tables (DLT), Machine Learning workloads, and SQL Warehouses** ✅ Correct

- Serverless supports:

  - Notebooks

  - Jobs

  - DLT

  - ML

  - SQL Warehouses

**D. Only Delta Live Tables (DLT)** ❌ Incorrect

- DLT is only one supported workload.

**E. Only Machine Learning workloads** ❌ Incorrect

- ML is only one supported workload.

**Key Concept**

Serverless spans nearly every major Databricks workload.
