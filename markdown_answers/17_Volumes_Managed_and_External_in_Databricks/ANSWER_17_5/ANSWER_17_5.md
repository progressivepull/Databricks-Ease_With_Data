# ANSWER 17 5

**Question 5**

**Which DBUtils command is used to create directories inside a volume?**

**A. dbutils.fs.cp()** ❌ Incorrect

- Copies files.

**B. dbutils.fs.head()** ❌ Incorrect

- Reads the beginning of a file.

**C. dbutils.fs.mkdirs()** ✅ Correct

Example:

dbutils.fs.mkdirs(\
"/Volumes/main/files/images"\
)

Creates folders just like:

mkdir

in Linux.

**D. dbutils.fs.mv()** ❌ Incorrect

- Moves files.

**E. dbutils.fs.rm()** ❌ Incorrect

- Removes files.

------------------------------------------------------------------------

**Memory Trick**

**mkdirs**

↓

Make Directories
