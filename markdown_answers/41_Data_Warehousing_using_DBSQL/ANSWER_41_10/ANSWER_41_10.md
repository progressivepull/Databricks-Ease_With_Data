# ANSWER 41 10

**Question 10**

**Which statement correctly compares SQL Warehouses with All-Purpose Clusters?**

**A. SQL Warehouses are optimized for analytics (OLAP), BI connectivity, and concurrent SQL workloads, while All-Purpose Clusters are designed primarily for engineering and development.** ✅ Correct

- This is the key distinction.

**SQL Warehouses**

- SQL analytics

- Dashboards

- BI tools

- High concurrency

- Photon optimization

- Reporting

**All-Purpose Clusters**

- Data engineering

- Python

- Scala

- Spark development

- Interactive notebooks

- ETL pipelines

**B. All-Purpose Clusters provide better BI integration than SQL Warehouses.** ❌ Incorrect

- SQL Warehouses are specifically designed for BI connectivity.

**C. SQL Warehouses cannot execute SQL queries from notebooks.** ❌ Incorrect

- SQL notebooks can use SQL Warehouses.

**D. All-Purpose Clusters always use the Photon Engine by default.** ❌ Incorrect

- Photon availability depends on cluster configuration and runtime; it is not universally enabled by default.

**E. SQL Warehouses are intended only for machine learning workloads.** ❌ Incorrect

- SQL Warehouses are built for analytics, not machine learning.

**Key Concept**

| **SQL Warehouse** | **All-Purpose Cluster**    |
|-------------------|----------------------------|
| SQL Analytics     | Data Engineering           |
| BI Tools          | Notebook Development       |
| Dashboards        | ETL Pipelines              |
| High Concurrency  | Interactive Spark          |
| OLAP              | General Spark Computing    |
| Photon Optimized  | Multi-language Development |
