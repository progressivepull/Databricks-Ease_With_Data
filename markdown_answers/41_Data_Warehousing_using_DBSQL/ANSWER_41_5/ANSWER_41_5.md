# ANSWER 41 5

**Question 5**

**What is the primary benefit of Predictive I/O in Serverless SQL Warehouses?**

**A. Automatic SQL code generation** ❌ Incorrect

- Predictive I/O doesn't write SQL.

**B. Faster data scanning and reduced query latency** ✅ Correct

- Predictive I/O intelligently reads only the data needed.

- Benefits include:

  - Less disk I/O

  - Faster scans

  - Lower latency

  - Better overall query performance

**C. Automatic dashboard creation** ❌ Incorrect

- Dashboard creation is a user task.

**D. Encryption of Delta tables** ❌ Incorrect

- Encryption is handled separately.

**E. Automatic schema evolution** ❌ Incorrect

- Schema evolution is a Delta Lake feature.

**Key Concept**

**Predictive I/O = Smarter data reading = Faster queries**
