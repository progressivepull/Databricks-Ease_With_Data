# ANSWER 41 9

**Question 9**

**What capability does the SQL Editor provide for saved queries?**

**A. Converting SQL into Python automatically** ❌ Incorrect

- No automatic SQL-to-Python conversion exists.

**B. Scheduling queries to run automatically for recurring reports and dashboards** ✅ Correct

- Users can:

  - Save queries

  - Schedule them

  - Refresh dashboards

  - Produce recurring reports

This supports automated reporting.

**C. Executing only DDL statements** ❌ Incorrect

- SQL Editor supports many SQL statement types.

**D. Managing Spark cluster configurations** ❌ Incorrect

- Cluster management is outside the SQL Editor.

**E. Automatically creating Unity Catalog schemas** ❌ Incorrect

- Schemas are created through SQL commands or catalog management.

**Key Concept**

**Saved Queries → Scheduled Execution → Automated Reports**
