# ANSWER 41 3

**Question 3**

**Which type of SQL Warehouse is recommended for most workloads?**

**A. Classic** ❌ Incorrect

- Classic is the older implementation.

- It lacks many newer performance improvements.

**B. Pro** ❌ Incorrect

- Pro adds features beyond Classic.

- However, Databricks generally recommends Serverless when available.

**C. Serverless** ✅ Correct

- Recommended because:

  - Fast startup

  - Automatic scaling

  - No cluster management

  - Lower operational overhead

  - Better performance

**D. Dedicated** ❌ Incorrect

- Dedicated isn't the recommended default SQL Warehouse type.

**E. Community** ❌ Incorrect

- No Community SQL Warehouse type exists.

**Key Concept**

**Recommended SQL Warehouse = Serverless**
