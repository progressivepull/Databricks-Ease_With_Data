# ANSWER 06 2

**Question 2**

**Which of the following items can users store in their personal workspace folders?**

**A. Only Delta tables** ❌ Incorrect

- Delta tables are stored in cloud storage and managed by Unity Catalog or the Hive metastore.

- They are not stored in Workspace folders.

**B. Only notebooks and SQL warehouses** ❌ Incorrect

- SQL Warehouses are compute resources.

- Workspace folders hold many more object types than notebooks.

**C. Notebooks, queries, dashboards, MLflow experiments, and files** ✅ **Correct**

- Personal workspace folders can contain:

  - Notebooks

  - SQL queries

  - Dashboards

  - MLflow experiments

  - Uploaded files

  - Git repositories (depending on configuration)

**D. Only Spark clusters and jobs** ❌ Incorrect

- Clusters are compute resources.

- Jobs are scheduled workloads.

- Neither is stored as a file inside Workspace folders.

**E. Only Unity Catalog metadata** ❌ Incorrect

- Unity Catalog metadata is stored in the metastore, not Workspace folders.

**Key Concept**

Workspace folders organize your **development assets**, not your data or compute resources.
