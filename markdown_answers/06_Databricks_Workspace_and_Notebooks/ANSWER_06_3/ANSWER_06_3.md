# ANSWER 06 3

**Question 3**

**Which programming languages are natively supported in Databricks notebooks?**

**A. Python, Java, C#, and Go** ❌ Incorrect

- Java, C#, and Go are not native notebook languages.

**B. Python, SQL, Scala, and R** ✅ **Correct**\
Databricks notebooks natively support:

- Python

- SQL

- Scala

- R

These four languages can even be mixed within the same notebook.

**C. JavaScript, SQL, Kotlin, and R** ❌ Incorrect

- JavaScript and Kotlin are not native notebook languages.

**D. Python, Ruby, Scala, and Julia** ❌ Incorrect

- Ruby and Julia are not officially supported notebook languages.

**E. SQL, Bash, Java, and PowerShell** ❌ Incorrect

- Bash can be executed using %sh, but it is not considered a primary notebook language.

- Java and PowerShell are not notebook languages.

**Key Concept**

Remember the four official notebook languages:

- Python

- SQL

- Scala

- R
