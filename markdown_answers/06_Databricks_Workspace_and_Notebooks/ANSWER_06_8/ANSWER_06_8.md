# ANSWER 06 8

**Question 8**

**What is the primary purpose of Markdown (Text) cells in a Databricks notebook?**

**A. Execute Spark jobs faster** ❌ Incorrect

- Markdown cells do not execute code.

**B. Store Delta transaction logs** ❌ Incorrect

- Delta logs are stored in the \_delta_log directory, not in notebooks.

**C. Document and explain notebook content without executing code** ✅ **Correct**

- Markdown cells are used to:

  - Explain code

  - Add headings

  - Create documentation

  - Include images and links

  - Improve notebook readability

**D. Create Compute clusters automatically** ❌ Incorrect

- Markdown has no effect on cluster creation.

**E. Configure Unity Catalog permissions** ❌ Incorrect

- Permissions are managed through Unity Catalog, not Markdown.

**Key Concept**

Good notebooks combine **code** with **documentation**, making them easier to understand and maintain.
