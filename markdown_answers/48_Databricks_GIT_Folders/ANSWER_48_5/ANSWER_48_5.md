# ANSWER 48 5

**Question 5**

**What is the purpose of the Sparse Checkout feature?**

**A. To automatically merge Git branches** ❌ Incorrect

- Sparse Checkout does not perform merges.

**B. To clone only selected folders from a repository, reducing the local workspace size** ✅ Correct

Suppose a repository contains:

Repository

Analytics

Engineering

Machine Learning

Infrastructure

Documentation

If you only need:

Engineering

Sparse Checkout clones only that folder.

Benefits:

- Faster clone

- Smaller workspace

- Less unnecessary code

**C. To convert notebooks into Delta tables** ❌ Incorrect

- Git has nothing to do with Delta tables.

**D. To back up notebooks to Unity Catalog Volumes** ❌ Incorrect

- Sparse Checkout is unrelated to Volumes.

**E. To encrypt Git repositories before cloning** ❌ Incorrect

- Encryption isn't the purpose.

**Key Concept**

**Sparse Checkout = Clone only what you need.**
