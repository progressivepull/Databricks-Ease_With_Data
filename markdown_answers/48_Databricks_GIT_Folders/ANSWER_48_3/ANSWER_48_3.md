# ANSWER 48 3

**Question 3**

**Which Azure DevOps service is used to store notebooks, source code, and configuration files using Git version control?**

**A. Boards** ❌ Incorrect

- Boards manage:

  - Work items

  - Tasks

  - Bugs

  - User stories

**B. Test Plans** ❌ Incorrect

- Test Plans manage software testing.

**C. Repos** ✅ Correct

- Azure DevOps Repos provide Git repositories.

- They store:

  - Notebooks

  - Python

  - SQL

  - YAML

  - Configuration files

**D. Artifacts** ❌ Incorrect

- Artifacts store packages such as NuGet or Maven packages.

**E. Monitor** ❌ Incorrect

- Azure Monitor is used for monitoring applications and infrastructure.

**Key Concept**

Azure DevOps Services

| **Service** | **Purpose**      |
|-------------|------------------|
| Boards      | Work tracking    |
| **Repos**   | Git repositories |
| Pipelines   | CI/CD            |
| Artifacts   | Package storage  |
| Test Plans  | Testing          |
