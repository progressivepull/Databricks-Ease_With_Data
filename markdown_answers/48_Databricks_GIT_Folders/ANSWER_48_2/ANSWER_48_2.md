# ANSWER 48 2

**Question 2**

**Which Git operation is supported by Databricks Git Folders?**

**A. Clone repositories** ✅ Correct

- One of the first Git operations is **Clone**.

- Cloning copies a remote repository into your Databricks workspace.

Example:

Azure DevOps

Repository

│

Clone

▼

Git Folder

Other supported Git operations include:

- Pull

- Commit

- Push

- Branch switching

- Merge (through Git workflows)

**B. Create SQL Warehouses** ❌ Incorrect

- SQL Warehouses are compute resources.

- Git doesn't create compute.

**C. Refresh Materialized Views** ❌ Incorrect

- Materialized Views are database objects.

**D. Create Delta tables** ❌ Incorrect

- Delta tables are created with SQL or Spark.

**E. Manage Unity Catalog permissions** ❌ Incorrect

- Permissions are managed through Unity Catalog.

**Key Concept**

**Git operations manage code, not compute or data objects.**
