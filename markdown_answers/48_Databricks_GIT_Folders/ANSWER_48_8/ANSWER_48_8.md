# ANSWER 48 8

**Question 8**

**After modifying a notebook, what indication does Databricks display before the changes are committed?**

**A. Ready (R)** ❌ Incorrect

- No such Git status indicator is used.

**B. Modified (M)** ✅ Correct

Example:

Notebook.py

M

The **M** means:

Changes exist locally.

Not yet committed.

Typical workflow:

Edit

│

M

│

Commit

│

Push

**C. Synced (S)** ❌ Incorrect

- Not used.

**D. Locked (L)** ❌ Incorrect

- Git doesn't lock notebooks.

**E. Published (P)** ❌ Incorrect

- Git doesn't use this status.

**Key Concept**

**M = Modified = Local changes waiting to be committed.**
