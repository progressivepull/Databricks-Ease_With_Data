# ANSWER 48 9

**Question 9**

**Which sequence correctly describes the Git workflow demonstrated in the lesson?**

**A. Pull → Commit → Clone → Push** ❌ Incorrect

- You clone before you can commit or push.

**B. Create Notebook → Edit Notebook → Commit → Push** ✅ Correct

Typical workflow:

Clone Repository

│

Create Notebook

│

Edit Notebook

│

Commit

│

Push

The quiz focuses on the work after the repository is already available.

**C. Push → Clone → Edit → Commit** ❌ Incorrect

- You can't push before cloning and making changes.

**D. Create Branch → Delete Repository → Commit → Pull** ❌ Incorrect

- Deleting the repository is unrelated.

**E. Create SQL Warehouse → Commit → Push → Publish Dashboard** ❌ Incorrect

- SQL Warehouses are unrelated to Git workflow.

**Key Concept**

**Create → Edit → Commit → Push**
