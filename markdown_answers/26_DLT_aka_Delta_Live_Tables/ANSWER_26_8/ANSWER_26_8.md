# ANSWER 26 8

**Question 8**

**Which Delta Live Tables edition includes both Change Data Capture (CDC) and Data Quality Expectations?**

**A. Community Edition ❌ Incorrect**

No such DLT edition.

**B. Core ❌ Incorrect**

Core provides basic pipeline capabilities but not the full advanced feature set.

**C. Standard ❌ Incorrect**

Does not include both advanced CDC and comprehensive data quality features.

**D. Pro ❌ Incorrect**

Pro adds additional capabilities but does not include the complete advanced feature set described in the lesson.

**E. Advanced ✅ Correct**

Advanced includes features such as:

- Change Data Capture (CDC)

- Data Quality Expectations

- Advanced pipeline capabilities

These are intended for enterprise-grade data engineering.

**Why E is correct**

Advanced is the most feature-rich DLT edition and includes both CDC and Data Quality Expectations.
