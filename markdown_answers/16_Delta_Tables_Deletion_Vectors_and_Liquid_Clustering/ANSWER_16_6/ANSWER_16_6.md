# ANSWER 16 6

**Question 6**

**Which Delta Lake and Databricks Runtime versions are required to use Deletion Vectors?**

**A. Delta Lake 1.0+ and DBR 10.4 LTS+** ❌ Incorrect

- Too old.

- Deletion Vectors didn't exist.

**B. Delta Lake 2.0+ and DBR 11.0+** ❌ Incorrect

- Still introduced before the feature.

**C. Delta Lake 2.3+ and Databricks Runtime 12.2 LTS+** ✅ Correct

Deletion Vectors require:

- Delta Lake **2.3+**

- Databricks Runtime **12.2 LTS+**

These versions introduced support.

**D. Delta Lake 3.1+ and DBR 13.3 LTS+ only** ❌ Incorrect

- These versions support Deletion Vectors but are not the minimum required versions.

**E. Any Delta Lake version with Photon enabled** ❌ Incorrect

- Photon alone is insufficient.
