# ANSWER 39 5

**Question 5**

**In Delta Sharing, what is the role of the Provider?**

**A. The organization that receives shared data ❌ Incorrect**

That's the **Recipient**.

**B. The service that authenticates Spark clusters ❌ Incorrect**

Authentication is handled separately.

**C. The organization that owns the data, creates shares, and grants recipients access ✅ Correct**

Provider responsibilities:

- owns data

- creates Shares

- adds tables

- grants access

- manages permissions

Example:

Provider

│

Creates Share

│

Adds Tables

│

Grants Recipient

**D. The catalog created by the recipient ❌ Incorrect**

Catalogs belong to recipients.

**E. The application that consumes Open Sharing tokens ❌ Incorrect**

That's the client application.

**Why C is correct**

The Provider is the data owner responsible for creating and managing shared datasets.
