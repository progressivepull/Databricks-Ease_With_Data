# ANSWER 49 5

**Question 5**

**Which authentication method is best suited for local development and interactive use?**

**A. Machine-to-Machine (M2M) using a Service Principal** ❌ Incorrect

- M2M is designed for automation.

- It does not require a human user.

**B. User-to-Machine (U2M) using a user account** ✅ Correct

During development:

Developer

↓

CLI

↓

Interactive Login

↓

Workspace

This allows developers to:

- Test commands

- Develop notebooks

- Deploy manually

**C. API Key authentication only** ❌ Incorrect

- The lesson focuses on U2M and M2M authentication.

**D. SSH key authentication** ❌ Incorrect

- SSH keys are unrelated to Databricks CLI authentication in this context.

**E. Anonymous authentication** ❌ Incorrect

- Databricks does not allow anonymous access.

**Key Concept**

Development

↓

**U2M**

Automation

↓

**M2M**
