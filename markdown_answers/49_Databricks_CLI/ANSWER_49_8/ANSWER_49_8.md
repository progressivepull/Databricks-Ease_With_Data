# ANSWER 49 8

**Question 8**

**Which command lists all Unity Catalog catalogs available to the authenticated user?**

**A. databricks schemas list** ❌ Incorrect

- Lists schemas, not catalogs.

**B. databricks account catalogs list** ❌ Incorrect

- Not the correct CLI syntax.

**C. databricks catalogs list** ✅ Correct

Example:

databricks catalogs list

Returns:

main

sales

finance

marketing

**D. databricks workspace catalogs** ❌ Incorrect

- Invalid syntax.

**E. databricks unity list** ❌ Incorrect

- Not a valid command.

**Key Concept**

**Catalogs → databricks catalogs list**
