# ANSWER 49 3

**Question 3**

**Which of the following is a key benefit of using the Databricks CLI?**

**A. It eliminates the need for SQL Warehouses.** ❌ Incorrect

- SQL Warehouses still execute SQL queries.

- The CLI does not replace compute.

**B. It enables automation, scripting, and CI/CD workflows.** ✅ Correct

This is why most engineers use the CLI.

Instead of manually clicking:

Create Job

Upload Notebook

Deploy Pipeline

You can automate everything.

Example:

databricks bundle deploy

or

databricks jobs create

Benefits:

- Automation

- Repeatability

- CI/CD

- Faster deployments

**C. It automatically converts all notebooks into Python scripts.** ❌ Incorrect

- The CLI does not convert notebook formats.

**D. It replaces Git repositories for source control.** ❌ Incorrect

- Git still manages source code.

**E. It manages Azure subscriptions directly.** ❌ Incorrect

- Azure CLI manages Azure subscriptions.

**Key Concept**

**CLI = Automation + CI/CD**
