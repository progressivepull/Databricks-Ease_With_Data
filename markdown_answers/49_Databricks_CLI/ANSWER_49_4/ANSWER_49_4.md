# ANSWER 49 4

**Question 4**

**Which command verifies that the Databricks CLI has been installed successfully?**

**A. databricks --status** ❌ Incorrect

- Not a valid CLI command.

**B. databricks version** ❌ Incorrect

- Not the command demonstrated in the lesson.

**C. databricks -v** ✅ Correct

This prints the installed CLI version.

Example:

databricks -v

Output:

Databricks CLI version x.x.x

If this works:

CLI is installed correctly.

**D. databricks verify** ❌ Incorrect

- Not a valid command.

**E. databricks check** ❌ Incorrect

- Not supported.

**Key Concept**

**databricks -v = Verify installation.**
