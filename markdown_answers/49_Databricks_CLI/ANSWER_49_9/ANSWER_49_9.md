# ANSWER 49 9

**Question 9**

**Which command displays help information for Databricks CLI commands and subcommands?**

**A. databricks info** ❌ Incorrect

- Not the general help command.

**B. databricks docs** ❌ Incorrect

- Not valid syntax.

**C. databricks --help** ✅ Correct

Example:

databricks --help

Shows:

- Commands

- Subcommands

- Usage

- Options

You can also use:

databricks catalogs --help

to see help for a specific command group.

**D. databricks guide** ❌ Incorrect

- Invalid.

**E. databricks tutorial** ❌ Incorrect

- Not supported.

**Key Concept**

**--help works for almost every CLI command.**
