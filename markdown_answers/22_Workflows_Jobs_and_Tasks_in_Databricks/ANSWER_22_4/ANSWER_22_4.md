**Question 4**

**Where can the source code for a notebook task be stored?**

**A. Only in the Databricks Workspace** ❌ Incorrect

- Workspace storage is supported.

- It isn't the only option.

**B. Only in Git repositories** ❌ Incorrect

- Git is also supported, but not exclusively.

**C. Either in the Databricks Workspace or a Git repository** ✅ Correct

Many organizations develop notebooks in Git.

Example:

Others keep notebooks inside Databricks.

Both approaches work.

**D. Only in Azure Blob Storage** ❌ Incorrect

- Blob Storage stores files, not notebook source locations.

**E. Only in DBFS** ❌ Incorrect

- DBFS isn't the recommended notebook source location.
