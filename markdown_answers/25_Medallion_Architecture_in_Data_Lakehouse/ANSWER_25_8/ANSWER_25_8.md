# ANSWER 25 8

**Question 8**

**Which file detection method is the default for Auto Loader?**

**A. Event Hub Notifications ❌ Incorrect**

Optional notification service.

Not default.

------------------------------------------------------------------------

**B. File Notification ❌ Incorrect**

Supported but must be enabled.

------------------------------------------------------------------------

**C. Kafka Streaming ❌ Incorrect**

Kafka is unrelated to cloud file discovery.

------------------------------------------------------------------------

**D. Directory Listing ✅ Correct**

Default behavior:

Cloud Storage

↓

List Directory

↓

Find New Files

↓

Load Them

No cloud messaging configuration is required.

------------------------------------------------------------------------

**E. Delta Change Data Feed ❌ Incorrect**

CDF tracks changes in Delta tables.

It does not discover new files.

------------------------------------------------------------------------

**Why D is correct**

Directory Listing is Auto Loader's default file discovery method.

------------------------------------------------------------------------
