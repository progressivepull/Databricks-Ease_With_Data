**Question 7**

**Which DBUtils command retrieves the current value of a widget?**

**A. dbutils.widgets.value()** ❌ Incorrect

- No such method.

**B. dbutils.widgets.get()** ✅ Correct

Example:

country =\
dbutils.widgets.get("country")

If the widget contains:

USA

Then:

country == "USA"

- Invalid command.
