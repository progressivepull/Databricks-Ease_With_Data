# ANSWER 18 2

**Question 2**

**Which DBUtils command displays all available DBUtils modules?**

**A. dbutils.fs.help()** ❌ Incorrect

- Shows help only for the **File System** module.

Example:

dbutils.fs.help()

Shows:

- cp

- ls

- rm

- mv

- mkdirs

- etc.

**B. dbutils.modules()** ❌ Incorrect

- No such command exists.

**C. dbutils.help()** ✅ Correct

Example:

dbutils.help()

Shows every DBUtils module, such as:

- fs

- widgets

- notebook

- secrets

- jobs

- library

- credentials (depending on environment)

**D. dbutils.list()** ❌ Incorrect

- Invalid command.

**E. dbutils.show()** ❌ Incorrect

- Invalid command.

**Memory Trick**

Need help for everything?

↓

dbutils.help()

Need help for files?

↓

dbutils.fs.help()
