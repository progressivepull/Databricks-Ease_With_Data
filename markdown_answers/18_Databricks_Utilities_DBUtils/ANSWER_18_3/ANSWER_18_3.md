# ANSWER 18 3

**Question 3**

**Which DBUtils module is primarily used for file and storage operations?**

**A. dbutils.widgets** ❌ Incorrect

- Widgets collect user input.

**B. dbutils.secrets** ❌ Incorrect

- Secrets retrieve passwords and API keys.

**C. dbutils.fs** ✅ Correct

This is the File System module.

Common commands:

dbutils.fs.ls()\
\
dbutils.fs.cp()\
\
dbutils.fs.mv()\
\
dbutils.fs.rm()\
\
dbutils.fs.mkdirs()\
\
dbutils.fs.head()

It works with:

- Unity Catalog Volumes

- DBFS

- Cloud Storage

**D. dbutils.notebook** ❌ Incorrect

- Executes notebooks.

**E. dbutils.jobs** ❌ Incorrect

- Used with Databricks Jobs.
