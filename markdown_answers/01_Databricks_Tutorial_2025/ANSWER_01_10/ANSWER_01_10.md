# ANSWER 01 10

Which of the following is a recommended prerequisite for learners before starting the Databricks series?

A. JavaScript\
B. Tableau\
C. Kubernetes\
**D. Python**\
E. Power BI

**Correct Answer:** D. Python

------------------------------------------------------------------------

**Why Each Option Is Right or Wrong**

**A. JavaScript ❌**

**Why it's wrong:**\
JavaScript is primarily used for web development (front-end and back-end applications). The course prerequisites specifically mention:

- Apache Spark

- PySpark

- Spark Streaming

- SQL

- Python

JavaScript is not listed as a recommended prerequisite and is not a core technology for Databricks data engineering workflows.

**Key association:** Web development, not Databricks data engineering.

------------------------------------------------------------------------

**B. Tableau ❌**

**Why it's wrong:**\
Tableau is a business intelligence (BI) and data visualization tool. While Databricks can integrate with Tableau for reporting and dashboards, knowledge of Tableau is not listed as a prerequisite for the series.

The course focuses on learning Databricks itself rather than external BI tools.

**Key association:** Data visualization and reporting.

------------------------------------------------------------------------

**C. Kubernetes ❌**

**Why it's wrong:**\
Kubernetes is a container orchestration platform used to manage containerized applications. Although it is important in cloud-native environments, it is not mentioned among the recommended prerequisites.

Databricks users typically interact with managed clusters rather than directly managing Kubernetes infrastructure.

**Key association:** Container orchestration and infrastructure management.

------------------------------------------------------------------------

**D. Python ✅**

**Why it's correct:**\
The instructor explicitly recommends having basic knowledge of:

- Apache Spark

- PySpark

- Spark Streaming

- SQL

- **Python**

Python is one of the most commonly used languages in Databricks for:

- Data engineering

- Data analysis

- Machine learning

- Writing notebooks

- Working with PySpark

Because it is directly listed in the prerequisites, this is the correct answer.

**Key association:** Core programming language for Databricks and PySpark.

------------------------------------------------------------------------

**E. Power BI ❌**

**Why it's wrong:**\
Power BI is another business intelligence and visualization platform. Similar to Tableau, it can connect to Databricks but is not identified as a prerequisite for the course.

The course focuses on building foundational Databricks skills rather than BI dashboard development.

**Key association:** Business intelligence and reporting.

------------------------------------------------------------------------

**Memory Tip**

Think of the prerequisites as the **Databricks technology stack**:

✅ **Python** → Programming language\
✅ **SQL** → Querying data\
✅ **Apache Spark** → Distributed processing\
✅ **PySpark** → Spark using Python\
✅ **Spark Streaming** → Real-time data processing

If you see options like **JavaScript, Tableau, Kubernetes, or Power BI**, they're related technologies but **not listed as prerequisites** in the course description. The only listed prerequisite among the options is **Python**.
