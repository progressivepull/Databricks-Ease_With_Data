# ANSWER 01 3

**What is a primary purpose of a data warehouse?**

A. To store raw, unprocessed data from multiple sources\
B. To handle real-time transactional processing for applications\
**C. To provide a centralized repository optimized for querying and analysis**\
D. To replace operational databases used in day-to-day business operations\
E. To store only unstructured data such as images and videos

**Correct Answer:** C

------------------------------------------------------------------------

**✅ Correct Answer: C**

**“To provide a centralized repository optimized for querying and analysis”**

This is the core purpose of a **data warehouse**:

- Centralizes data from multiple sources

- Cleans and structures the data

- Optimized for:

  - Reporting

  - Business intelligence (BI)

  - Analytics

👉 Designed for **fast, reliable querying and decision-making**

------------------------------------------------------------------------

**❌ A. To store raw, unprocessed data from multiple sources**

**Why wrong:**

- This describes a **data lake**, not a data warehouse

- Data warehouses store:

  - Cleaned ✅

  - Transformed ✅

  - Structured data ✅

👉 Not raw data

------------------------------------------------------------------------

**❌ B. To handle real-time transactional processing for applications**

**Why wrong:**

- This is the role of **OLTP systems** (operational databases)

- Examples:

  - Banking systems

  - E-commerce transactions

Data warehouses use **OLAP (analytical processing)** instead

👉 Not built for transactions

------------------------------------------------------------------------

**❌ D. To replace operational databases used in day-to-day business operations**

**Why wrong:**

- Data warehouses **do not replace** operational systems

- They **complement** them:

  - Operational DB → handles daily transactions

  - Data warehouse → analyzes historical data

👉 Different purposes

------------------------------------------------------------------------

**❌ E. To store only unstructured data such as images and videos**

**Why wrong:**

- Data warehouses primarily store:

  - Structured data (tables, rows, columns)

- Not designed for unstructured data like:

  - Images

  - Videos

👉 That’s more aligned with **data lakes**

------------------------------------------------------------------------

**🔑 Simple Summary**

- **Data Warehouse = Clean + Structured + Analytics-focused**

- Built for **querying and reporting**, not raw storage or transactions

- That’s why **C is correct**
