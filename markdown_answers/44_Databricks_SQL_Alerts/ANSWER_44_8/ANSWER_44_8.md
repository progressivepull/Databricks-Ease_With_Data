# ANSWER 44 8

**Question 8**

**Which of the following is NOT a supported notification destination for Databricks SQL Alerts?**

**A. Slack** ❌ Incorrect

- Slack is supported.

**B. Microsoft Teams** ❌ Incorrect

- Microsoft Teams is supported.

**C. PagerDuty** ❌ Incorrect

- PagerDuty is supported for operational incident management.

**D. Email** ❌ Incorrect

- Email is one of the most common notification methods.

**E. GitHub Issues** ✅ Correct

- GitHub Issues is **not** a built-in notification destination.

- Alerts cannot automatically create GitHub Issues.

**Key Concept**

Supported examples:

- Email

- Slack

- Microsoft Teams

- PagerDuty

Not supported:

- GitHub Issues
