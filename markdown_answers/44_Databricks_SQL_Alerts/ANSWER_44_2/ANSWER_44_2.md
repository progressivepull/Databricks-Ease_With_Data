# ANSWER 44 2

**Question 2**

**Which of the following is a common use case for Databricks SQL Alerts?**

**A. Creating Unity Catalog metastores** ❌ Incorrect

- Unity Catalog administration has nothing to do with Alerts.

**B. Monitoring pending order counts, data quality issues, and business KPIs** ✅ Correct

- SQL Alerts are commonly used to monitor:

  - Pending orders

  - Sales KPIs

  - Revenue thresholds

  - Inventory levels

  - Failed ETL jobs

  - Data quality issues

Example:

Pending Orders

Current = 15,000

Threshold = 10,000

Alert Sent

**C. Converting CSV files into Delta tables** ❌ Incorrect

- COPY INTO or Auto Loader performs ingestion.

**D. Managing Spark cluster autoscaling** ❌ Incorrect

- Cluster management is unrelated.

**E. Deploying machine learning models** ❌ Incorrect

- MLflow handles model deployment.

**Key Concept**

**SQL Alerts monitor business metrics and operational health.**
