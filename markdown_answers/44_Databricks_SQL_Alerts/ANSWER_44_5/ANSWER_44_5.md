# ANSWER 44 5

**Question 5**

**Which comparison operator is supported when configuring a SQL Alert trigger condition?**

**A. Greater than (\>)** ✅ Correct

- SQL Alerts compare returned values using operators such as:

  - 

  - \<

  - ≥

  - ≤

  - =

  - ≠ (depending on the interface)

Example:

Pending Orders \> 10,000

If true:

Trigger Alert

**B. GROUP BY** ❌ Incorrect

- GROUP BY is a SQL clause.

- It is not a comparison operator.

**C. JOIN** ❌ Incorrect

- JOIN combines tables.

**D. UNION** ❌ Incorrect

- UNION combines query results.

**E. ORDER BY** ❌ Incorrect

- ORDER BY sorts results.

**Key Concept**

**Alert conditions use comparison operators, not SQL clauses.**
