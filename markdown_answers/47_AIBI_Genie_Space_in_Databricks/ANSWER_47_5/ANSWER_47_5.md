# ANSWER 47 5

**Question 5**

**What is the purpose of configuring Example Values in a Genie Space?**

**A. To increase SQL Warehouse performance** ❌ Incorrect

- Example values don't make SQL faster.

**B. To improve AI interpretation of user questions by providing representative data values** ✅ Correct

Suppose a column contains

NE

SE

MW

AI might not know these mean

- Northeast

- Southeast

- Midwest

Example Values teach Genie.

Likewise,

Status

A

P

C

Example Values explain:

- Active

- Pending

- Closed

This improves natural-language understanding.

**C. To automatically create dashboard filters** ❌ Incorrect

- Filters are separate dashboard features.

**D. To replace dataset descriptions** ❌ Incorrect

- Dataset descriptions and example values complement each other.

**E. To generate Materialized Views automatically** ❌ Incorrect

- Genie doesn't create Materialized Views.

**Key Concept**

**Example Values teach the AI what real business data looks like.**
