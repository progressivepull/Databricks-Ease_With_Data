# ANSWER 47 8

**Question 8**

**Which capability does Genie provide in addition to returning query results?**

**A. Automatic cluster provisioning only** ❌ Incorrect

- Genie focuses on analytics, not compute provisioning.

**B. Automatic visualizations such as bar charts, pie charts, and trend charts** ✅ Correct

Instead of only returning:

| **Region** | **Sales** |
|------------|-----------|
| East       | 100       |
| West       | 120       |

Genie may automatically generate:

📊 Bar Chart

📈 Trend Chart

🥧 Pie Chart

This makes insights easier to understand.

**C. Automatic creation of Delta Live Tables pipelines** ❌ Incorrect

- Not a Genie feature.

**D. Automatic generation of Python notebooks** ❌ Incorrect

- Genie generates SQL, not notebooks.

**E. Automatic creation of SQL Alerts** ❌ Incorrect

- Alerts must be created separately.

**Key Concept**

**Genie returns both data and visualizations.**
