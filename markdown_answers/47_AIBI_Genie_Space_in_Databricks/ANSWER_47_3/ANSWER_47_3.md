# ANSWER 47 3

**Question 3**

**Which SQL Warehouse types are supported by AI/BI Genie?**

**A. Classic SQL Warehouse only** ❌ Incorrect

- Classic is not the only supported option.

**B. SQL Warehouse Pro and SQL Warehouse Serverless** ✅ Correct

Genie supports:

- SQL Warehouse Pro

- SQL Warehouse Serverless

These warehouses execute the generated SQL.

**C. All-purpose compute only** ❌ Incorrect

- Genie doesn't use All-Purpose Clusters.

**D. Job clusters only** ❌ Incorrect

- Job clusters run scheduled jobs.

**E. Photon clusters only** ❌ Incorrect

- Photon is an optimization engine, not a warehouse type.

**Key Concept**

**Genie runs on SQL Warehouses—not notebook clusters.**
