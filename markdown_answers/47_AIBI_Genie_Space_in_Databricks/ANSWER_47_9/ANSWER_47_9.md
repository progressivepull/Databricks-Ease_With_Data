# ANSWER 47 9

**Question 9**

**What information is available in the Monitoring tab of a Genie Space?**

**A. Only SQL Warehouse costs** ❌ Incorrect

- Costs aren't the primary focus.

**B. Only Unity Catalog permissions** ❌ Incorrect

- Permissions are managed elsewhere.

**C. Questions asked, generated SQL, users, and query history** ✅ Correct

Administrators can review:

- User questions

- Generated SQL

- Query history

- Users

- AI behavior

This helps improve Genie.

Example

User:

"Show last month's sales"

Generated SQL:

SELECT...

Execution Time:

1.4 sec

**D. Only dashboard refresh schedules** ❌ Incorrect

- Genie isn't focused on dashboard schedules.

**E. Only cluster utilization metrics** ❌ Incorrect

- Cluster monitoring is separate.

**Key Concept**

**Monitoring = Understand how people are using Genie.**
