# ANSWER 43 5

**Question 5**

**Which of the following databases is supported by Lakehouse Federation?**

**A. PostgreSQL** ✅ Correct

- PostgreSQL is one of the supported external databases.

Other supported systems include:

- PostgreSQL

- MySQL

- SQL Server

- Snowflake

- Redshift

- BigQuery (depending on platform support)

**B. MongoDB only** ❌ Incorrect

- The question asks for a supported relational database.

- "Only" makes the statement incorrect.

**C. Apache Kafka only** ❌ Incorrect

- Kafka is a streaming platform, not a relational database.

**D. Delta Lake only** ❌ Incorrect

- Delta Lake is native Databricks storage, not an external federation source.

**E. Unity Catalog only** ❌ Incorrect

- Unity Catalog is governance metadata, not an external database.

**Key Concept**

**Lakehouse Federation supports several external relational databases, including PostgreSQL.**
