# ANSWER 43 6

**Question 6**

**Which compute environment can be used to test and run Lakehouse Federation connections?**

**A. SQL Warehouses and Classic Databricks Compute** ✅ Correct

- Federation works from:

  - SQL Warehouses

  - Classic Compute (clusters)

- This allows both SQL analysts and Spark developers to query external data.

**B. Unity Catalog only** ❌ Incorrect

- Unity Catalog stores metadata.

- It does not execute queries.

**C. Delta Sharing only** ❌ Incorrect

- Delta Sharing is unrelated.

**D. MLflow Tracking Server only** ❌ Incorrect

- MLflow manages machine learning experiments.

**E. Databricks Repos only** ❌ Incorrect

- Repos store source code.

- They do not execute SQL queries.

**Key Concept**

**Compute executes queries; Unity Catalog manages metadata.**
