# ANSWER 34 9

**Question 9**

**What additional capabilities does the MANAGE privilege provide?**

**A. Querying tables only ❌ Incorrect**

Querying requires:

SELECT

**B. Reading files from Volumes only ❌ Incorrect**

Requires:

READ VOLUME

**C. Renaming objects, dropping objects, and changing permissions ✅ Correct**

MANAGE provides administrative capabilities.

Examples include:

- rename tables

- rename schemas

- drop objects

- grant permissions

- revoke permissions

Think of MANAGE as:

"Administrator of this object."

**D. Creating Spark clusters automatically ❌ Incorrect**

Clusters are unrelated.

**E. Running notebooks as another user ❌ Incorrect**

MANAGE does not impersonate users.

**Why C is correct**

MANAGE controls administration of Unity Catalog objects, including object maintenance and permission management.
