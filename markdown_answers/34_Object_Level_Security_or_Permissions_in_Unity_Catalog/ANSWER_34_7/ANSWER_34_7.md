# ANSWER 34 7

**Question 7**

**Which privilege allows a user to read files stored in a Unity Catalog Volume?**

**A. SELECT ❌ Incorrect**

SELECT is for tables and views.

Volumes store files.

**B. MODIFY ❌ Incorrect**

MODIFY changes table data.

**C. READ VOLUME ✅ Correct**

Volumes store files such as:

CSV

PDF

Images

Models

ZIP files

Reading those files requires:

READ VOLUME

**D. WRITE VOLUME ❌ Incorrect**

Allows writing files.

Not reading.

**E. EXECUTE ❌ Incorrect**

EXECUTE applies to functions or procedures.

**Why C is correct**

READ VOLUME specifically grants permission to read files stored in Unity Catalog Volumes.
