# ANSWER 23 2

**Question 2**

**Which of the following file formats is supported by the COPY INTO command?**

**A. CSV only** ❌ Incorrect

- CSV is supported, but many other formats are supported as well.

**B. CSV, JSON, Parquet, Avro, Text, and Binary files** ✅ Correct

COPY INTO supports multiple formats including:

- CSV

- JSON

- Parquet

- Avro

- Text

- Binary

This flexibility allows ingestion of many different data sources.

**C. Delta files only** ❌ Incorrect

- COPY INTO is primarily designed to ingest external file formats into Delta tables.

**D. XML only** ❌ Incorrect

- XML is not the only supported format.

**E. Excel (.xlsx) only** ❌ Incorrect

- Excel files are not directly supported by COPY INTO.
