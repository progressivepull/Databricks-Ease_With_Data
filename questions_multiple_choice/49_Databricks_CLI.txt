Question 1
What is the primary purpose of the Databricks CLI (Command Line Interface)?
A. To replace Unity Catalog for data governance
B. To manage Databricks accounts and workspaces using terminal commands
C. To automatically create SQL Warehouses
D. To generate AI-powered dashboards
E. To convert notebooks into Delta tables
Correct Answer: B. To manage Databricks accounts and workspaces using terminal commands

Question 2
The Databricks CLI is built on top of which underlying technology?
A. Apache Spark SQL Engine
B. Databricks REST API
C. Delta Live Tables (DLT)
D. Unity Catalog Metadata Service
E. Azure Resource Manager (ARM) Templates
