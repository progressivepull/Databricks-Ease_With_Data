Question 1
What is the primary purpose of Auto Loader in Databricks?
A. To manually copy files from local machines into DBFS
B. To efficiently and incrementally ingest new files from cloud storage into Delta Lake tables
C. To replace Unity Catalog metadata management
D. To optimize SQL query performance on Delta tables
E. To create Spark clusters automatically
Correct Answer: B. To efficiently and incrementally ingest new files from cloud storage into Delta Lake tables

Question 3
How does Auto Loader achieve exactly-once processing?

B. By storing processed file metadata in a checkpoint directory using RocksDB
C. By renaming processed files with a timestamp
