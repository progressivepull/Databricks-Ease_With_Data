Question 1
What is the primary purpose of Row-Level Security (RLS) in Unity Catalog?
A. To prevent users from accessing an entire Databricks workspace
B. To encrypt sensitive columns in a table
C. To restrict which rows each user can view within the same table
D. To automatically create separate tables for each user
E. To hide SQL query text from users
Correct Answer: C. To restrict which rows each user can view within the same table
