Question 1
What is the primary purpose of Column-Level Masking in Unity Catalog?
A. To remove sensitive columns from a table permanently
B. To restrict access to entire rows based on user identity
C. To dynamically hide or mask sensitive column values while keeping the column visible
D. To encrypt all data stored in Delta Lake
E. To create separate tables for different user groups
Correct Answer: C. To dynamically hide or mask sensitive column values while keeping the column visible

Question 2
How does Column-Level Masking differ from Row-Level Security?
A. Column-Level Masking hides entire rows, while Row-Level Security hides columns.
B. Column-Level Masking protects individual column values, while Row-Level Security controls which rows are visible.
C. Both features perform exactly the same function.
D. Row-Level Security works only with SQL, while Column-Level Masking works only with PySpark.
