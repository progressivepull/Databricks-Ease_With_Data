Question 1
What is the primary purpose of Databricks SQL Alerts?
A. To automatically optimize SQL queries for better performance
B. To continuously monitor SQL query results and send notifications when specified conditions are met
C. To automatically create Materialized Views from SQL queries
D. To replicate SQL query results into Delta Lake
